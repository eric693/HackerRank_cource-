#49 MEDIUM

Group Anagrams

Array Hash Table String Sorting

題目描述

給定一個字串陣列 strs，將字母異位詞組合在一起。可以按任意順序返回結果列表。

字母異位詞是由重新排列源單詞的所有字母得到的一個新單詞。

範例

範例 1:

輸入：strs = ["eat","tea","tan","ate","nat","bat"]
輸出：[["bat"],["nat","tan"],["ate","eat","tea"]]

範例 2:

輸入：strs = [""]
輸出：[[""]]

範例 3:

輸入：strs = ["a"]
輸出：[["a"]]

限制條件

  • 1 <= strs.length <= 104
  • 0 <= strs[i].length <= 100
  • strs[i] 僅包含小寫英文字母

解題思路

核心思想

字母異位詞的特徵：包含相同的字母，只是順序不同。

方法一：排序 + 哈希表（推薦）

將排序後的字串作為鍵：

  1. 對每個字串排序
  2. 排序後的字串作為哈希表的鍵
  3. 相同鍵的字串分為一組
  • 時間複雜度：O(n × k log k) - n 是字串數量，k 是字串長度
  • 空間複雜度：O(n × k)

方法二：字符計數 + 哈希表

用字符頻率作為鍵：

  • 統計每個字母出現的次數
  • 將計數陣列轉為元組作為鍵
  • 時間複雜度：O(n × k)
  • 空間複雜度：O(n × k)

程式碼實作

Python 解法

from collections import defaultdict

class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        """
        排序 + 哈希表
        
        時間複雜度：O(n × k log k)
        空間複雜度：O(n × k)
        """
        anagrams = defaultdict(list)
        
        for s in strs:
            # 排序後的字串作為鍵
            key = ''.join(sorted(s))
            anagrams[key].append(s)
        
        return list(anagrams.values())


# 方法二：字符計數
class Solution2:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        """
        字符計數 + 哈希表
        
        時間複雜度：O(n × k)
        空間複雜度：O(n × k)
        """
        anagrams = defaultdict(list)
        
        for s in strs:
            # 統計字符頻率
            count = [0] * 26
            for char in s:
                count[ord(char) - ord('a')] += 1
            
            # 將計數陣列轉為元組作為鍵
            key = tuple(count)
            anagrams[key].append(s)
        
        return list(anagrams.values())


# 方法三：更簡潔的寫法
class Solution3:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        """
        簡潔版排序法
        """
        anagrams = {}
        
        for s in strs:
            key = ''.join(sorted(s))
            if key not in anagrams:
                anagrams[key] = []
            anagrams[key].append(s)
        
        return list(anagrams.values())

C++ 解法

#include <vector>
#include <string>
#include <unordered_map>
#include <algorithm>
using namespace std;

class Solution {
public:
    vector<vector<string>> groupAnagrams(vector<string>& strs) {
        unordered_map<string, vector<string>> anagrams;
        
        for (const string& s : strs) {
            string key = s;
            sort(key.begin(), key.end());
            anagrams[key].push_back(s);
        }
        
        vector<vector<string>> result;
        for (auto& pair : anagrams) {
            result.push_back(pair.second);
        }
        
        return result;
    }
};

Java 解法

import java.util.*;

class Solution {
    public List<List<String>> groupAnagrams(String[] strs) {
        Map<String, List<String>> anagrams = new HashMap<>();
        
        for (String s : strs) {
            char[] chars = s.toCharArray();
            Arrays.sort(chars);
            String key = new String(chars);
            
            anagrams.computeIfAbsent(key, k -> new ArrayList<>()).add(s);
        }
        
        return new ArrayList<>(anagrams.values());
    }
}

面試要點

常見追問

  • 能否優化到 O(n × k)？
  • 如何處理大小寫？
  • 如何處理 Unicode 字符？
  • 兩種方法的優劣比較？

解題技巧

  • 排序法最直觀易懂
  • 字符計數法更高效
  • 哈希表分組是關鍵
  • 理解字母異位詞的本質

常見錯誤

  • 忘記處理空字串
  • 字符計數陣列大小錯誤
  • 鍵的生成方式不一致
  • 返回格式錯誤

算法比較

方法一：排序

"eat" → 排序 → "aet"
"tea" → 排序 → "aet"  ← 相同鍵
"tan" → 排序 → "ant"
"ate" → 排序 → "aet"  ← 相同鍵
"nat" → 排序 → "ant"  ← 相同鍵
"bat" → 排序 → "abt"

分組結果：
"aet": ["eat", "tea", "ate"]
"ant": ["tan", "nat"]
"abt": ["bat"]

方法二：字符計數

"eat" → [1,0,0,0,1,0,...,1] (a:1, e:1, t:1)
"tea" → [1,0,0,0,1,0,...,1] ← 相同鍵
"bat" → [1,1,0,0,0,0,...,1] (a:1, b:1, t:1)

優點：O(k) 時間生成鍵（不需要排序）
缺點：空間稍大（26 個字母的計數）
