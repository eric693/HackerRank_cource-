#102 MEDIUM

Binary Tree Level Order Traversal

Tree BFS Binary Tree

題目描述

給定一個二元樹的根節點 root，返回其節點值的層序遍歷。（即逐層地，從左到右訪問所有節點）。

範例

範例 1:

輸入：root = [3,9,20,null,null,15,7]
輸出：[[3],[9,20],[15,7]]

樹的結構：
    3
   / \
  9  20
    /  \
   15   7

範例 2:

輸入：root = [1]
輸出：[[1]]

範例 3:

輸入：root = []
輸出：[]

限制條件

  • 樹中節點數目在範圍 [0, 2000] 內
  • -1000 <= Node.val <= 1000

解題思路

BFS（廣度優先搜尋）- 推薦

使用佇列（Queue）實現層序遍歷：

  1. 將根節點加入佇列
  2. 對每一層：記錄當前層的節點數量
  3. 將當前層的所有節點依序取出，並將它們的子節點加入佇列
  4. 重複直到佇列為空
  • 時間複雜度：O(n) - 每個節點訪問一次
  • 空間複雜度：O(n) - 佇列最多存儲一層的節點

DFS（深度優先搜尋）

使用遞迴搭配層級參數，將節點按層級加入對應的列表。

  • 時間複雜度：O(n)
  • 空間複雜度：O(n) - 遞迴堆疊

程式碼實作

Python 解法

from collections import deque
from typing import Optional, List

class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution:
    def levelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
        """
        BFS 解法 - 使用佇列
        
        時間複雜度：O(n)
        空間複雜度：O(n)
        """
        if not root:
            return []
        
        result = []
        queue = deque([root])
        
        while queue:
            level_size = len(queue)
            current_level = []
            
            # 處理當前層的所有節點
            for _ in range(level_size):
                node = queue.popleft()
                current_level.append(node.val)
                
                # 將下一層的節點加入佇列
                if node.left:
                    queue.append(node.left)
                if node.right:
                    queue.append(node.right)
            
            result.append(current_level)
        
        return result


# 方法二：DFS 解法
class Solution2:
    def levelOrder(self, root: Optional[TreeNode]) -> List[List[int]]:
        """
        DFS 解法 - 使用遞迴
        
        時間複雜度：O(n)
        空間複雜度：O(n)
        """
        result = []
        
        def dfs(node, level):
            if not node:
                return
            
            # 如果這是新的一層，創建新的列表
            if len(result) == level:
                result.append([])
            
            # 將當前節點的值加入對應層級
            result[level].append(node.val)
            
            # 遞迴處理左右子樹
            dfs(node.left, level + 1)
            dfs(node.right, level + 1)
        
        dfs(root, 0)
        return result

C++ 解法

#include <vector>
#include <queue>
using namespace std;

struct TreeNode {
    int val;
    TreeNode *left;
    TreeNode *right;
    TreeNode() : val(0), left(nullptr), right(nullptr) {}
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
    TreeNode(int x, TreeNode *left, TreeNode *right) : val(x), left(left), right(right) {}
};

class Solution {
public:
    vector<vector<int>> levelOrder(TreeNode* root) {
        vector<vector<int>> result;
        if (!root) return result;
        
        queue<TreeNode*> q;
        q.push(root);
        
        while (!q.empty()) {
            int levelSize = q.size();
            vector<int> currentLevel;
            
            // 處理當前層的所有節點
            for (int i = 0; i < levelSize; i++) {
                TreeNode* node = q.front();
                q.pop();
                currentLevel.push_back(node->val);
                
                // 將下一層的節點加入佇列
                if (node->left) q.push(node->left);
                if (node->right) q.push(node->right);
            }
            
            result.push_back(currentLevel);
        }
        
        return result;
    }
};

Java 解法

import java.util.*;

class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;
    TreeNode() {}
    TreeNode(int val) { this.val = val; }
    TreeNode(int val, TreeNode left, TreeNode right) {
        this.val = val;
        this.left = left;
        this.right = right;
    }
}

class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> result = new ArrayList<>();
        if (root == null) return result;
        
        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);
        
        while (!queue.isEmpty()) {
            int levelSize = queue.size();
            List<Integer> currentLevel = new ArrayList<>();
            
            // 處理當前層的所有節點
            for (int i = 0; i < levelSize; i++) {
                TreeNode node = queue.poll();
                currentLevel.add(node.val);
                
                // 將下一層的節點加入佇列
                if (node.left != null) queue.offer(node.left);
                if (node.right != null) queue.offer(node.right);
            }
            
            result.add(currentLevel);
        }
        
        return result;
    }
}

面試要點

常見追問

  • 如何實現之字形（zigzag）層序遍歷？
  • 如何從右到左遍歷每一層？
  • 能否用 DFS 實現？差異是什麼？
  • 如何找出每層的最大值？

解題技巧

  • BFS 是層序遍歷的標準解法
  • 記錄每層的節點數量是關鍵
  • 使用佇列（Queue）資料結構
  • DFS 也可以實現，但不如 BFS 直觀

常見錯誤

  • 忘記記錄每層的節點數量
  • 沒有檢查空節點
  • 混淆 BFS 和 DFS 的實現
  • 佇列操作順序錯誤
