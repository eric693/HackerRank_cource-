#155 MEDIUM

Min Stack

Stack Design

題目描述

設計一個支援 push、pop、top 操作，並能在常數時間內檢索到最小元素的堆疊。

實現 MinStack 類別：

  • MinStack() 初始化堆疊物件。
  • void push(int val) 將元素 val 推入堆疊。
  • void pop() 刪除堆疊頂端的元素。
  • int top() 獲取堆疊頂端的元素。
  • int getMin() 獲取堆疊中的最小元素。

範例

範例 1:

輸入
["MinStack","push","push","push","getMin","pop","top","getMin"]
[[],[-2],[0],[-3],[],[],[],[]]

輸出
[null,null,null,null,-3,null,0,-2]

解釋
MinStack minStack = new MinStack();
minStack.push(-2);
minStack.push(0);
minStack.push(-3);
minStack.getMin(); // 返回 -3
minStack.pop();
minStack.top();    // 返回 0
minStack.getMin(); // 返回 -2

限制條件

  • -231 <= val <= 231 - 1
  • pop、top 和 getMin 操作總是在非空堆疊上調用
  • push、pop、top 和 getMin 最多被調用 3 * 104 次

解題思路

核心挑戰

如何在 O(1) 時間內獲取最小值？

方法一：雙堆疊（推薦）

使用兩個堆疊：

  • 主堆疊：存儲所有元素
  • 最小堆疊：存儲當前的最小值

當 push 時，如果新元素 ≤ 當前最小值，也推入最小堆疊。

  • 時間複雜度：O(1) - 所有操作
  • 空間複雜度：O(n)

方法二：單堆疊 + 配對

每個元素與當前最小值配對存儲。

  • 時間複雜度：O(1)
  • 空間複雜度：O(n)

方法三：差值法（進階）

存儲與最小值的差值，可以節省空間。

程式碼實作

Python 解法

class MinStack:
    """
    雙堆疊解法
    
    時間複雜度：O(1) - 所有操作
    空間複雜度：O(n)
    """
    
    def __init__(self):
        # 主堆疊存儲所有元素
        self.stack = []
        # 最小堆疊存儲對應的最小值
        self.min_stack = []

    def push(self, val: int) -> None:
        self.stack.append(val)
        
        # 如果最小堆疊為空或新值更小，推入最小堆疊
        if not self.min_stack or val <= self.min_stack[-1]:
            self.min_stack.append(val)

    def pop(self) -> None:
        if self.stack:
            val = self.stack.pop()
            # 如果彈出的是最小值，也從最小堆疊彈出
            if val == self.min_stack[-1]:
                self.min_stack.pop()

    def top(self) -> int:
        return self.stack[-1] if self.stack else None

    def getMin(self) -> int:
        return self.min_stack[-1] if self.min_stack else None


# 方法二：單堆疊 + 配對
class MinStack2:
    """
    每個元素與最小值配對
    """
    
    def __init__(self):
        # 存儲 (值, 當前最小值) 的元組
        self.stack = []

    def push(self, val: int) -> None:
        if not self.stack:
            self.stack.append((val, val))
        else:
            current_min = min(val, self.stack[-1][1])
            self.stack.append((val, current_min))

    def pop(self) -> None:
        if self.stack:
            self.stack.pop()

    def top(self) -> int:
        return self.stack[-1][0] if self.stack else None

    def getMin(self) -> int:
        return self.stack[-1][1] if self.stack else None


# 方法三：差值法（進階）
class MinStack3:
    """
    存儲與最小值的差值
    """
    
    def __init__(self):
        self.stack = []
        self.min_val = None

    def push(self, val: int) -> None:
        if not self.stack:
            self.stack.append(0)
            self.min_val = val
        else:
            # 存儲差值
            self.stack.append(val - self.min_val)
            if val < self.min_val:
                self.min_val = val

    def pop(self) -> None:
        if self.stack:
            diff = self.stack.pop()
            # 如果差值為負，需要恢復之前的最小值
            if diff < 0:
                self.min_val = self.min_val - diff

    def top(self) -> int:
        if self.stack:
            diff = self.stack[-1]
            if diff < 0:
                return self.min_val
            else:
                return self.min_val + diff
        return None

    def getMin(self) -> int:
        return self.min_val

C++ 解法

#include <stack>
using namespace std;

class MinStack {
private:
    stack<int> mainStack;
    stack<int> minStack;
    
public:
    MinStack() {}
    
    void push(int val) {
        mainStack.push(val);
        
        if (minStack.empty() || val <= minStack.top()) {
            minStack.push(val);
        }
    }
    
    void pop() {
        if (!mainStack.empty()) {
            int val = mainStack.top();
            mainStack.pop();
            
            if (val == minStack.top()) {
                minStack.pop();
            }
        }
    }
    
    int top() {
        return mainStack.top();
    }
    
    int getMin() {
        return minStack.top();
    }
};

Java 解法

import java.util.Stack;

class MinStack {
    private Stack<Integer> mainStack;
    private Stack<Integer> minStack;
    
    public MinStack() {
        mainStack = new Stack<>();
        minStack = new Stack<>();
    }
    
    public void push(int val) {
        mainStack.push(val);
        
        if (minStack.isEmpty() || val <= minStack.peek()) {
            minStack.push(val);
        }
    }
    
    public void pop() {
        if (!mainStack.isEmpty()) {
            int val = mainStack.pop();
            
            if (val == minStack.peek()) {
                minStack.pop();
            }
        }
    }
    
    public int top() {
        return mainStack.peek();
    }
    
    public int getMin() {
        return minStack.peek();
    }
}

面試要點

常見追問

  • 如何實現 Max Stack？
  • 能否只用一個堆疊？
  • 如何處理相同的最小值？
  • 空間複雜度能否優化？

解題技巧

  • 雙堆疊是最直觀的解法
  • 注意相等情況的處理
  • 配對法更易理解但空間略大
  • 差值法最優但較複雜

常見錯誤

  • 忘記處理相等的最小值
  • pop 時沒有同步最小堆疊
  • 空堆疊檢查遺漏
  • 比較條件用 < 而非 ≤

視覺化說明

雙堆疊操作過程：

操作序列：push(-2), push(0), push(-3), getMin(), pop()

主堆疊        最小堆疊
  -2            -2        push(-2)
  -2  0         -2        push(0)  // 0 > -2，不推入
  -2  0  -3     -2  -3    push(-3) // -3 ≤ -2，推入
  
getMin() → -3 (最小堆疊頂)

  -2  0         -2        pop()    // -3 == -3，兩邊都彈出
  
getMin() → -2 (最小堆疊頂)
