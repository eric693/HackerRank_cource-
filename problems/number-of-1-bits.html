#191 EASY

Number of 1 Bits

Bit Manipulation Divide and Conquer

題目描述

編寫一個函數，輸入是一個無符號整數（以二進制形式），返回其二進制表達式中數字位數為 '1' 的個數（也被稱為漢明重量）。

範例

範例 1:

輸入：n = 00000000000000000000000000001011
輸出：3
解釋：輸入的二進制串 00000000000000000000000000001011 中，共有三位為 '1'。

範例 2:

輸入：n = 00000000000000000000000010000000
輸出：1
解釋：輸入的二進制串 00000000000000000000000010000000 中，共有一位為 '1'。

範例 3:

輸入：n = 11111111111111111111111111111101
輸出：31
解釋：輸入的二進制串 11111111111111111111111111111101 中，共有 31 位為 '1'。

限制條件

  • 輸入必須是長度為 32 的二進制串

解題思路

方法一：Brian Kernighan 算法（推薦）

每次消除最右邊的 1：

  • n & (n - 1) 會將最右邊的 1 變為 0
  • 重複操作直到 n 變為 0
  • 操作次數 = 1 的個數
  • 時間複雜度：O(k) - k 為 1 的個數
  • 空間複雜度：O(1)

方法二：逐位檢查

檢查每一位是否為 1：

  • 使用 n & 1 檢查最低位
  • 右移 n 繼續檢查
  • 時間複雜度：O(32) = O(1)
  • 空間複雜度：O(1)

方法三：內建函數

使用語言內建的位元計數函數。

程式碼實作

Python 解法

class Solution:
    def hammingWeight(self, n: int) -> int:
        """
        Brian Kernighan 算法
        
        時間複雜度：O(k) - k 為 1 的個數
        空間複雜度：O(1)
        """
        count = 0
        while n:
            n &= n - 1  # 消除最右邊的 1
            count += 1
        return count


# 方法二：逐位檢查
class Solution2:
    def hammingWeight(self, n: int) -> int:
        """
        逐位檢查
        
        時間複雜度：O(32) = O(1)
        空間複雜度：O(1)
        """
        count = 0
        for i in range(32):
            if n & (1 << i):  # 檢查第 i 位是否為 1
                count += 1
        return count


# 方法三：右移檢查
class Solution3:
    def hammingWeight(self, n: int) -> int:
        """
        右移檢查
        """
        count = 0
        while n:
            count += n & 1  # 檢查最低位
            n >>= 1         # 右移
        return count


# 方法四：內建函數
class Solution4:
    def hammingWeight(self, n: int) -> int:
        """
        使用內建函數
        """
        return bin(n).count('1')

C++ 解法

Brian Kernighan 算法原理

為什麼 n & (n-1) 能消除最右邊的 1？

以 n = 12 (二進制: 1100) 為例：

n     = 1100
n-1   = 1011  (借位導致最右邊的 1 變 0，右邊的 0 都變 1)
n&(n-1)= 1000  (消除了最右邊的 1)

再次操作：
n     = 1000
n-1   = 0111
n&(n-1)= 0000  (消除了最後一個 1)

共操作 2 次 = 2 個 1

過程視覺化

n = 11 (二進制: 1011)

第 1 次：1011 & 1010 = 1010 (消除最右邊的 1)
第 2 次：1010 & 1001 = 1000 (消除最右邊的 1)
第 3 次：1000 & 0111 = 0000 (消除最後一個 1)

共 3 次操作 = 3 個 1

位元操作技巧總結

常用位元操作

  • n & (n-1)：消除最右邊的 1
  • n & 1：檢查最低位是否為 1（判斷奇偶）
  • n | (1 << i)：將第 i 位設為 1
  • n & ~(1 << i)：將第 i 位設為 0
  • n ^ (1 << i)：翻轉第 i 位
  • n >> 1：右移一位（除以 2）
  • n << 1：左移一位（乘以 2）
