#238 MEDIUM

Product of Array Except Self

Array Prefix Sum

題目描述

給定一個整數陣列 nums，返回一個陣列 answer，使得 answer[i] 等於 nums 中除 nums[i] 之外其餘各元素的乘積。

題目保證陣列 nums 中任意元素的全部前綴元素和後綴元素的乘積都在 32 位整數範圍內。

請不要使用除法，且在 O(n) 時間複雜度內完成此題。

範例

範例 1:

輸入：nums = [1,2,3,4]
輸出：[24,12,8,6]

範例 2:

輸入：nums = [-1,1,0,-3,3]
輸出：[0,0,9,0,0]

限制條件

  • 2 <= nums.length <= 105
  • -30 <= nums[i] <= 30
  • 保證陣列 nums 中任意元素的全部前綴元素和後綴元素的乘積都在 32 位整數範圍內

進階要求

你可以在 O(1) 的額外空間複雜度內完成這個題目嗎？（輸出陣列不被視為額外空間）

解題思路

左右乘積列表（推薦）

關鍵觀察：answer[i] = 左邊所有數的乘積 × 右邊所有數的乘積

  1. 先計算每個位置左邊所有數的乘積
  2. 再計算每個位置右邊所有數的乘積
  3. 將兩者相乘得到答案
  • 時間複雜度：O(n)
  • 空間複雜度：O(n) - 可優化到 O(1)

空間優化版

直接在輸出陣列上操作，省去額外空間。

  • 時間複雜度：O(n)
  • 空間複雜度：O(1)

程式碼實作

Python 解法

class Solution:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        """
        空間優化版 - O(1) 額外空間
        
        時間複雜度：O(n)
        空間複雜度：O(1)
        """
        n = len(nums)
        answer = [1] * n
        
        # 第一次遍歷：計算左邊的乘積
        left_product = 1
        for i in range(n):
            answer[i] = left_product
            left_product *= nums[i]
        
        # 第二次遍歷：計算右邊的乘積並相乘
        right_product = 1
        for i in range(n - 1, -1, -1):
            answer[i] *= right_product
            right_product *= nums[i]
        
        return answer


# 方法二：使用額外陣列（更好理解）
class Solution2:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        """
        使用左右陣列
        
        時間複雜度：O(n)
        空間複雜度：O(n)
        """
        n = len(nums)
        
        # 左邊乘積陣列
        left = [1] * n
        for i in range(1, n):
            left[i] = left[i-1] * nums[i-1]
        
        # 右邊乘積陣列
        right = [1] * n
        for i in range(n - 2, -1, -1):
            right[i] = right[i+1] * nums[i+1]
        
        # 結合左右陣列
        answer = [left[i] * right[i] for i in range(n)]
        
        return answer


# 方法三：詳細註解版
class Solution3:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        """
        詳細步驟說明
        
        例如：nums = [1, 2, 3, 4]
        
        左邊乘積：[1, 1, 2, 6]
        (每個位置左邊所有數的乘積)
        
        右邊乘積：[24, 12, 4, 1]
        (每個位置右邊所有數的乘積)
        
        最終結果：[24, 12, 8, 6]
        (左邊乘積 × 右邊乘積)
        """
        n = len(nums)
        answer = [1] * n
        
        # 計算左邊的累積乘積
        # answer[i] = nums[0] * nums[1] * ... * nums[i-1]
        left = 1
        for i in range(n):
            answer[i] = left
            left *= nums[i]
        
        # 計算右邊的累積乘積並相乘
        # answer[i] *= nums[i+1] * nums[i+2] * ... * nums[n-1]
        right = 1
        for i in range(n - 1, -1, -1):
            answer[i] *= right
            right *= nums[i]
        
        return answer

C++ 解法

#include <vector>
using namespace std;

class Solution {
public:
    vector<int> productExceptSelf(vector<int>& nums) {
        int n = nums.size();
        vector<int> answer(n, 1);
        
        // 計算左邊的乘積
        int leftProduct = 1;
        for (int i = 0; i < n; i++) {
            answer[i] = leftProduct;
            leftProduct *= nums[i];
        }
        
        // 計算右邊的乘積並相乘
        int rightProduct = 1;
        for (int i = n - 1; i >= 0; i--) {
            answer[i] *= rightProduct;
            rightProduct *= nums[i];
        }
        
        return answer;
    }
};

面試要點

常見追問

  • 為什麼不能使用除法？
  • 如何優化到 O(1) 空間？
  • 如果有 0 怎麼處理？
  • 能否用對數轉換？

解題技巧

  • 分解為「左邊乘積」和「右邊乘積」
  • 兩次遍歷即可完成
  • 直接在輸出陣列上操作節省空間
  • 前綴積（Prefix Product）的應用

常見錯誤

  • 使用除法（題目禁止）
  • 沒有正確初始化累積變數
  • 遍歷方向錯誤
  • 忘記處理邊界情況

視覺化說明

以 nums = [1, 2, 3, 4] 為例：

原陣列：     [1,  2,  3,  4]

左邊乘積：   [1,  1,  2,  6]
             ↑   ↑   ↑   ↑
             1   1   1×2 1×2×3

右邊乘積：   [24, 12, 4,  1]
             ↑   ↑   ↑   ↑
           2×3×4 3×4 4   1

最終結果：   [24, 12, 8,  6]
             ↑   ↑   ↑   ↑
            1×24 1×12 2×4 6×1
