#198 MEDIUM

House Robber

Array Dynamic Programming

題目描述

你是一個專業的小偷，計劃偷竊沿街的房屋。每間房內都藏有一定的現金，影響你偷竊的唯一制約因素就是相鄰的房屋裝有相互連通的防盜系統，如果兩間相鄰的房屋在同一晚上被小偷闖入，系統會自動報警。

給定一個代表每個房屋存放金額的非負整數陣列，計算你不觸動警報裝置的情況下，一夜之內能夠偷竊到的最高金額。

範例

範例 1:

輸入：nums = [1,2,3,1]
輸出：4
解釋：偷竊 1 號房屋 (金額 = 1) ，然後偷竊 3 號房屋 (金額 = 3)。
     偷竊到的最高金額 = 1 + 3 = 4。

範例 2:

輸入：nums = [2,7,9,3,1]
輸出：12
解釋：偷竊 1 號房屋 (金額 = 2), 偷竊 3 號房屋 (金額 = 9)，
     接著偷竊 5 號房屋 (金額 = 1)。
     偷竊到的最高金額 = 2 + 9 + 1 = 12。

限制條件

  • 1 <= nums.length <= 100
  • 0 <= nums[i] <= 400

解題思路

動態規劃

這是一個經典的動態規劃問題，關鍵在於理解狀態轉移：

  1. 狀態定義：dp[i] 表示到第 i 個房屋時能偷到的最大金額
  2. 狀態轉移：對於第 i 個房屋，有兩種選擇：
    • 偷：dp[i-2] + nums[i]
    • 不偷：dp[i-1]
  3. 轉移方程：dp[i] = max(dp[i-1], dp[i-2] + nums[i])
  • 時間複雜度：O(n)
  • 空間複雜度：O(1) - 可以優化到只用兩個變數

程式碼實作

Python 解法

class Solution:
    def rob(self, nums: List[int]) -> int:
        """
        動態規劃 - 空間優化版
        
        時間複雜度：O(n)
        空間複雜度：O(1)
        """
        if not nums:
            return 0
        if len(nums) == 1:
            return nums[0]
        
        # 只需要記錄前兩個狀態
        prev2 = nums[0]  # dp[i-2]
        prev1 = max(nums[0], nums[1])  # dp[i-1]
        
        for i in range(2, len(nums)):
            current = max(prev1, prev2 + nums[i])
            prev2 = prev1
            prev1 = current
        
        return prev1


# 方法二：使用陣列的 DP
class Solution2:
    def rob(self, nums: List[int]) -> int:
        """
        動態規劃 - 使用陣列
        
        時間複雜度：O(n)
        空間複雜度：O(n)
        """
        if not nums:
            return 0
        if len(nums) == 1:
            return nums[0]
        
        n = len(nums)
        dp = [0] * n
        dp[0] = nums[0]
        dp[1] = max(nums[0], nums[1])
        
        for i in range(2, n):
            dp[i] = max(dp[i-1], dp[i-2] + nums[i])
        
        return dp[n-1]


# 方法三：更簡潔的寫法
class Solution3:
    def rob(self, nums: List[int]) -> int:
        """
        最簡潔的寫法
        """
        prev = curr = 0
        
        for num in nums:
            prev, curr = curr, max(curr, prev + num)
        
        return curr

C++ 解法

#include <vector>
#include <algorithm>
using namespace std;

class Solution {
public:
    int rob(vector<int>& nums) {
        if (nums.empty()) return 0;
        if (nums.size() == 1) return nums[0];
        
        int prev2 = nums[0];
        int prev1 = max(nums[0], nums[1]);
        
        for (int i = 2; i < nums.size(); i++) {
            int current = max(prev1, prev2 + nums[i]);
            prev2 = prev1;
            prev1 = current;
        }
        
        return prev1;
    }
};

Java 解法

class Solution {
    public int rob(int[] nums) {
        if (nums.length == 0) return 0;
        if (nums.length == 1) return nums[0];
        
        int prev2 = nums[0];
        int prev1 = Math.max(nums[0], nums[1]);
        
        for (int i = 2; i < nums.length; i++) {
            int current = Math.max(prev1, prev2 + nums[i]);
            prev2 = prev1;
            prev1 = current;
        }
        
        return prev1;
    }
}

面試要點

常見追問

  • 如果房屋排列成環形怎麼辦？（House Robber II）
  • 如果房屋是二元樹結構？（House Robber III）
  • 如何輸出具體偷了哪些房屋？
  • 能否用遞迴實現？

解題技巧

  • 識別出這是動態規劃問題
  • 理解「偷」或「不偷」的選擇
  • 狀態轉移方程很關鍵
  • 空間可以優化到 O(1)

常見錯誤

  • 忘記處理邊界條件
  • 狀態轉移方程寫錯
  • 初始化錯誤
  • 沒有取 max 只取了其中一種情況

延伸問題

House Robber II（環形房屋）

房屋排列成環形，第一個和最後一個房屋相鄰。解法是分兩種情況：

  • 不考慮第一個房屋：rob(nums[1:n])
  • 不考慮最後一個房屋：rob(nums[0:n-1])
  • 取兩者的最大值

House Robber III（樹形房屋）

房屋排列成二元樹，相鄰節點不能同時偷。使用樹形 DP 解決。
