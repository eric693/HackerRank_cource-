#SQL-1 EASY

Employee Salary Query

SQL Database

題目描述

查詢員工薪資資訊，包含基本的 SELECT、WHERE、ORDER BY 操作。

資料表結構

Employees 表格：

欄位名稱 資料型態 說明
employee_id INT 員工編號（主鍵）
name VARCHAR(100) 員工姓名
department VARCHAR(50) 部門
salary DECIMAL(10,2) 薪資
hire_date DATE 到職日期

問題列表

問題 1：查詢所有員工資訊

查詢所有員工的姓名、部門和薪資。

SELECT name, department, salary
FROM Employees;

問題 2：查詢薪資大於 50000 的員工

找出薪資超過 50000 的所有員工。

SELECT name, department, salary
FROM Employees
WHERE salary > 50000;

問題 3：按薪資降序排列

查詢所有員工，按薪資從高到低排序。

SELECT name, department, salary
FROM Employees
ORDER BY salary DESC;

問題 4：查詢特定部門員工

查詢 Engineering 部門的所有員工。

SELECT name, salary, hire_date
FROM Employees
WHERE department = 'Engineering'
ORDER BY hire_date;

問題 5：薪資範圍查詢

查詢薪資在 40000 到 70000 之間的員工。

SELECT name, department, salary
FROM Employees
WHERE salary BETWEEN 40000 AND 70000
ORDER BY salary;

問題 6：多條件查詢

查詢 Sales 部門且薪資大於 45000 的員工。

SELECT name, salary
FROM Employees
WHERE department = 'Sales' AND salary > 45000
ORDER BY salary DESC;

問題 7：使用 LIKE 模糊查詢

查詢姓名以 'J' 開頭的員工。

SELECT name, department, salary
FROM Employees
WHERE name LIKE 'J%'
ORDER BY name;

問題 8：限制結果數量

查詢薪資最高的前 5 名員工。

SELECT name, department, salary
FROM Employees
ORDER BY salary DESC
LIMIT 5;

SQL 基礎語法回顧

1. SELECT 語句

SELECT column1, column2
FROM table_name;

2. WHERE 條件篩選

SELECT * FROM table_name
WHERE condition;

3. ORDER BY 排序

SELECT * FROM table_name
ORDER BY column_name ASC;  -- 升序
ORDER BY column_name DESC; -- 降序

4. LIMIT 限制結果

SELECT * FROM table_name
LIMIT 10;

5. 比較運算符

  • =：等於
  • != 或 <>：不等於
  • >：大於
  • <：小於
  • >=：大於等於
  • <=：小於等於
  • BETWEEN：在範圍內
  • IN：在列表中
  • LIKE：模糊匹配

6. 邏輯運算符

  • AND：且（所有條件都必須為真）
  • OR：或（至少一個條件為真）
  • NOT：非（條件取反）

面試要點

基礎概念

  • 理解 SELECT 語句結構
  • 掌握 WHERE 條件篩選
  • 熟悉 ORDER BY 排序
  • 了解 LIMIT 使用場景

常見錯誤

  • 忘記在字串值加引號
  • SQL 關鍵字大小寫混用
  • WHERE 條件邏輯錯誤
  • ORDER BY 忘記指定方向

優化技巧

  • 只查詢需要的欄位
  • 合理使用索引
  • 避免 SELECT *
  • WHERE 條件順序優化
