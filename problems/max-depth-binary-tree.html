#104 EASY

Maximum Depth of Binary Tree

Tree DFS BFS Binary Tree

題目描述

給定一個二元樹的根節點 root，返回其最大深度。

二元樹的最大深度是指從根節點到最遠葉子節點的最長路徑上的節點數。

範例

範例 1:

輸入：root = [3,9,20,null,null,15,7]
輸出：3

樹結構：
    3
   / \
  9  20
    /  \
   15   7

最大深度 = 3 (3 → 20 → 15 或 3 → 20 → 7)

範例 2:

輸入：root = [1,null,2]
輸出：2

樹結構：
  1
   \
    2

限制條件

  • 樹中節點的數量在範圍 [0, 104] 內
  • -100 <= Node.val <= 100

解題思路

方法一：遞迴 DFS（推薦）

最直觀的解法：

  1. 空節點深度為 0
  2. 非空節點深度 = 1 + max(左子樹深度, 右子樹深度)
  • 時間複雜度：O(n) - 訪問每個節點一次
  • 空間複雜度：O(h) - h 為樹的高度，遞迴堆疊

方法二：BFS（層序遍歷）

使用佇列進行層序遍歷：

  • 每處理一層，深度加 1
  • 最後返回總層數
  • 時間複雜度：O(n)
  • 空間複雜度：O(w) - w 為樹的最大寬度

方法三：DFS 迭代（堆疊）

使用堆疊模擬遞迴。

程式碼實作

Python 解法

class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        """
        遞迴 DFS - 最簡潔
        
        時間複雜度：O(n)
        空間複雜度：O(h)
        """
        if not root:
            return 0
        
        return 1 + max(self.maxDepth(root.left), self.maxDepth(root.right))


# 方法二：BFS（層序遍歷）
from collections import deque

class Solution2:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        """
        BFS 層序遍歷
        
        時間複雜度：O(n)
        空間複雜度：O(w)
        """
        if not root:
            return 0
        
        queue = deque([root])
        depth = 0
        
        while queue:
            depth += 1
            # 處理當前層的所有節點
            for _ in range(len(queue)):
                node = queue.popleft()
                if node.left:
                    queue.append(node.left)
                if node.right:
                    queue.append(node.right)
        
        return depth


# 方法三：DFS 迭代（堆疊）
class Solution3:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        """
        DFS 迭代版本
        
        時間複雜度：O(n)
        空間複雜度：O(h)
        """
        if not root:
            return 0
        
        stack = [(root, 1)]
        max_depth = 0
        
        while stack:
            node, depth = stack.pop()
            max_depth = max(max_depth, depth)
            
            if node.left:
                stack.append((node.left, depth + 1))
            if node.right:
                stack.append((node.right, depth + 1))
        
        return max_depth

C++ 解法

struct TreeNode {
    int val;
    TreeNode *left;
    TreeNode *right;
    TreeNode() : val(0), left(nullptr), right(nullptr) {}
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};

class Solution {
public:
    int maxDepth(TreeNode* root) {
        if (!root) return 0;
        
        return 1 + max(maxDepth(root->left), maxDepth(root->right));
    }
};

Java 解法

class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;
    TreeNode() {}
    TreeNode(int val) { this.val = val; }
}

class Solution {
    public int maxDepth(TreeNode root) {
        if (root == null) return 0;
        
        return 1 + Math.max(maxDepth(root.left), maxDepth(root.right));
    }
}

面試要點

常見追問

  • 如何求最小深度？
  • DFS 和 BFS 哪個更好？
  • 如何不用遞迴？
  • 時間空間複雜度分析？

解題技巧

  • 遞迴最簡潔直觀
  • 理解樹的高度定義
  • 空節點返回 0
  • BFS 適合層級問題

常見錯誤

  • 忘記處理空樹
  • 深度從 0 還是 1 開始
  • 遞迴終止條件錯誤
  • BFS 層數計算錯誤

遞迴過程視覺化

以範例 1 為例：

    3            maxDepth(3)
   / \             = 1 + max(left, right)
  9  20           = 1 + max(1, 2)
    /  \          = 3
   15   7

左子樹：
  9 → maxDepth(9) = 1 + max(0, 0) = 1

右子樹：
    20          maxDepth(20)
   /  \           = 1 + max(1, 1)
  15   7          = 2
  
  15 → maxDepth(15) = 1
  7  → maxDepth(7) = 1

BFS vs DFS

何時用 BFS？

  • 需要層級信息
  • 需要找最短路徑
  • 樹很深但不寬

何時用 DFS？

  • 代碼簡潔性優先
  • 樹很寬但不深
  • 需要遍歷所有節點

本題中：DFS 遞迴最簡潔，BFS 更直觀地體現「深度」概念。
