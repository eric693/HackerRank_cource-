#235 MEDIUM

Lowest Common Ancestor of a Binary Search Tree

Tree BST DFS Binary Tree

題目描述

給定一個二元搜尋樹（BST）的根節點 root，以及 BST 中的兩個節點 p 和 q，請找出這兩個節點的最低公共祖先（LCA）。

最低公共祖先的定義為：「對於有根樹 T 的兩個節點 p、q，最低公共祖先表示為一個節點 x，滿足 x 是 p、q 的祖先且 x 的深度盡可能大（一個節點也可以是它自己的祖先）。」

範例

範例 1:

輸入：root = [6,2,8,0,4,7,9,null,null,3,5], p = 2, q = 8
輸出：6
解釋：節點 2 和節點 8 的最低公共祖先是 6。

樹結構：
        6
       / \
      2   8
     / \ / \
    0  4 7  9
      / \
     3   5

範例 2:

輸入：root = [6,2,8,0,4,7,9,null,null,3,5], p = 2, q = 4
輸出：2
解釋：節點 2 和節點 4 的最低公共祖先是 2，因為根據定義最低公共祖先節點可以為節點本身。

範例 3:

輸入：root = [2,1], p = 2, q = 1
輸出：2

限制條件

  • 樹中節點數量在範圍 [2, 105] 內
  • -109 <= Node.val <= 109
  • 所有 Node.val 互不相同
  • p != q
  • p 和 q 均存在於給定的 BST 中

解題思路

利用 BST 的性質

BST 的關鍵特性：左子樹 < 根節點 < 右子樹

核心觀察

LCA 是 p 和 q 的分岔點：

  • 如果 p 和 q 都小於當前節點 → LCA 在左子樹
  • 如果 p 和 q 都大於當前節點 → LCA 在右子樹
  • 否則，當前節點就是 LCA（分岔點）

方法一：遞迴（推薦）

  • 時間複雜度：O(h) - h 為樹高
  • 空間複雜度：O(h) - 遞迴堆疊

方法二：迭代

  • 時間複雜度：O(h)
  • 空間複雜度：O(1)

程式碼實作

Python 解法

class TreeNode:
    def __init__(self, x):
        self.val = x
        self.left = None
        self.right = None

class Solution:
    def lowestCommonAncestor(self, root: TreeNode, p: TreeNode, q: TreeNode) -> TreeNode:
        """
        遞迴解法 - 利用 BST 性質
        
        時間複雜度：O(h)
        空間複雜度：O(h)
        """
        # 如果 p 和 q 都小於當前節點，LCA 在左子樹
        if p.val < root.val and q.val < root.val:
            return self.lowestCommonAncestor(root.left, p, q)
        
        # 如果 p 和 q 都大於當前節點，LCA 在右子樹
        if p.val > root.val and q.val > root.val:
            return self.lowestCommonAncestor(root.right, p, q)
        
        # 否則，當前節點就是 LCA（分岔點）
        return root


# 方法二：迭代
class Solution2:
    def lowestCommonAncestor(self, root: TreeNode, p: TreeNode, q: TreeNode) -> TreeNode:
        """
        迭代解法
        
        時間複雜度：O(h)
        空間複雜度：O(1)
        """
        current = root
        
        while current:
            # 都在左子樹
            if p.val < current.val and q.val < current.val:
                current = current.left
            # 都在右子樹
            elif p.val > current.val and q.val > current.val:
                current = current.right
            # 找到分岔點
            else:
                return current
        
        return None


# 方法三：更簡潔的寫法
class Solution3:
    def lowestCommonAncestor(self, root: TreeNode, p: TreeNode, q: TreeNode) -> TreeNode:
        """
        簡潔版遞迴
        """
        if (p.val < root.val and q.val < root.val):
            return self.lowestCommonAncestor(root.left, p, q)
        elif (p.val > root.val and q.val > root.val):
            return self.lowestCommonAncestor(root.right, p, q)
        else:
            return root

struct TreeNode {

class Solution {
public:
    TreeNode* lowestCommonAncestor(TreeNode* root, TreeNode* p, TreeNode* q) {

        if (p->val < root->val && q->val < root->val) {
            return lowestCommonAncestor(root->left, p, q);

        // 都在右子樹
        if (p->val > root->val && q->val > root->val) {
            return lowestCommonAncestor(root->right, p, q);
        }

        return root;
    }
};

面試要點

常見追問

  • 如果是普通二元樹怎麼做？
  • 能否用 O(1) 空間？
  • 如何處理節點不存在的情況？
  • BST 和普通樹的 LCA 有何不同？

解題技巧

  • 充分利用 BST 的有序性
  • 理解分岔點的概念
  • 迭代版本更節省空間
  • 節點可以是自己的祖先

常見錯誤

  • 忘記節點可以是自己的祖先
  • 比較邏輯錯誤
  • 沒有利用 BST 性質
  • 遞迴終止條件錯誤

算法視覺化

以範例 1 為例：p = 2, q = 8

        6  ← 當前節點
       / \
      2   8
     / \ / \
    0  4 7  9
      / \
     3   5

步驟：
1. 當前節點 = 6
   p(2) < 6 ? Yes
   q(8) < 6 ? No
   → 2 和 8 在 6 的兩側，6 就是 LCA ✓

---

範例 2：p = 2, q = 4

        6
       / \
      2  ← LCA
     / \
    0   4
       / \
      3   5

步驟：
1. 當前節點 = 6
   p(2) < 6 and q(4) < 6 → 往左走

2. 當前節點 = 2
   p(2) = 2 and q(4) > 2
   → 2 和 4 在不同側（一個等於當前節點）
   → 2 就是 LCA ✓

BST vs 普通二元樹

BST 的 LCA

  • 利用 BST 的有序性
  • 只需比較值的大小
  • 時間複雜度：O(h)
  • 空間複雜度：O(1) 可能

普通二元樹的 LCA

  • 需要遞迴搜索兩棵子樹
  • 更複雜的邏輯
  • 時間複雜度：O(n)
  • 空間複雜度：O(h)

BST 的優勢：利用有序性，更高效地找到 LCA。
