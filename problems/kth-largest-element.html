#215 MEDIUM

Kth Largest Element in an Array

Array Divide and Conquer Heap Quickselect

題目描述

給定整數陣列 nums 和整數 k，請返回陣列中第 k 個最大的元素。

請注意，你需要找的是陣列排序後的第 k 個最大的元素，而不是第 k 個不同的元素。

你必須設計並實現時間複雜度為 O(n) 的算法解決此問題。

範例

範例 1:

輸入：nums = [3,2,1,5,6,4], k = 2
輸出：5

範例 2:

輸入：nums = [3,2,3,1,2,4,5,5,6], k = 4
輸出：4

限制條件

  • 1 <= k <= nums.length <= 105
  • -104 <= nums[i] <= 104

解題思路

方法一：快速選擇（Quickselect）- 推薦

基於快速排序的分治算法：

  1. 選擇一個樞軸元素
  2. 將陣列分為三部分：大於、等於、小於樞軸
  3. 根據 k 的位置決定在哪一部分繼續搜索
  • 時間複雜度：O(n) 平均，O(n²) 最壞
  • 空間複雜度：O(1)

方法二：最小堆（Min Heap）

維護大小為 k 的最小堆：

  • 遍歷陣列，將元素加入堆
  • 如果堆大小超過 k，彈出最小元素
  • 最後堆頂元素就是第 k 大元素
  • 時間複雜度：O(n log k)
  • 空間複雜度：O(k)

方法三：排序（最簡單但不是最優）

  • 時間複雜度：O(n log n)
  • 空間複雜度：O(1) 或 O(n)

程式碼實作

Python 解法

import heapq
import random

class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        """
        快速選擇算法（Quickselect）
        
        時間複雜度：O(n) 平均
        空間複雜度：O(1)
        """
        def quickselect(left, right):
            # 隨機選擇樞軸避免最壞情況
            pivot_index = random.randint(left, right)
            pivot = nums[pivot_index]
            
            # 將樞軸移到最右邊
            nums[pivot_index], nums[right] = nums[right], nums[pivot_index]
            
            # 分區
            store_index = left
            for i in range(left, right):
                if nums[i] > pivot:
                    nums[i], nums[store_index] = nums[store_index], nums[i]
                    store_index += 1
            
            # 將樞軸放回正確位置
            nums[store_index], nums[right] = nums[right], nums[store_index]
            
            # 判斷樞軸位置
            if store_index == k - 1:
                return nums[store_index]
            elif store_index < k - 1:
                return quickselect(store_index + 1, right)
            else:
                return quickselect(left, store_index - 1)
        
        return quickselect(0, len(nums) - 1)


# 方法二：最小堆
class Solution2:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        """
        最小堆解法
        
        時間複雜度：O(n log k)
        空間複雜度：O(k)
        """
        # 維護大小為 k 的最小堆
        heap = []
        
        for num in nums:
            heapq.heappush(heap, num)
            if len(heap) > k:
                heapq.heappop(heap)
        
        # 堆頂就是第 k 大元素
        return heap[0]


# 方法三：使用 Python 內建函數
class Solution3:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        """
        使用 heapq.nlargest
        
        時間複雜度：O(n log k)
        """
        return heapq.nlargest(k, nums)[-1]


# 方法四：排序法（簡單但不是最優）
class Solution4:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        """
        排序法
        
        時間複雜度：O(n log n)
        """
        nums.sort(reverse=True)
        return nums[k - 1]

C++ 解法

#include <vector>
#include <queue>
#include <algorithm>
using namespace std;

class Solution {
public:
    int findKthLargest(vector<int>& nums, int k) {
        // 最小堆解法
        priority_queue<int, vector<int>, greater<int>> minHeap;
        
        for (int num : nums) {
            minHeap.push(num);
            if (minHeap.size() > k) {
                minHeap.pop();
            }
        }
        
        return minHeap.top();
    }
};

Java 解法

import java.util.PriorityQueue;

class Solution {
    public int findKthLargest(int[] nums, int k) {
        // 最小堆解法
        PriorityQueue<Integer> minHeap = new PriorityQueue<>();
        
        for (int num : nums) {
            minHeap.offer(num);
            if (minHeap.size() > k) {
                minHeap.poll();
            }
        }
        
        return minHeap.peek();
    }
}

面試要點

常見追問

  • 如何優化到 O(n) 時間？
  • 快速選擇和快速排序的區別？
  • 為什麼用最小堆而不是最大堆？
  • 如何處理重複元素？

解題技巧

  • 快速選擇是最優解法
  • 最小堆適合數據流場景
  • 隨機樞軸避免最壞情況
  • 理解分治的思想

常見錯誤

  • 混淆第 k 大和第 k 小
  • 分區邏輯錯誤
  • 堆的大小控制不當
  • 忘記處理邊界情況

快速選擇算法視覺化

以 nums = [3,2,1,5,6,4], k = 2 為例：

目標：找第 2 大元素

原陣列：[3, 2, 1, 5, 6, 4]

選擇樞軸 4，分區後：
[6, 5] 4 [3, 2, 1]
  大於    等於  小於

第 2 大在左邊部分（因為左邊有 2 個元素）

繼續在 [6, 5] 中找第 2 大：
選擇樞軸 5，分區後：
[6] 5 []

第 2 大就是 5 ✓

為什麼用最小堆？

維護大小為 k 的最小堆，堆頂是這 k 個元素中最小的：

例如找第 2 大，維護大小為 2 的最小堆：

遍歷 [3, 2, 1, 5, 6, 4]：

heap = [3]
heap = [2, 3]
heap = [2, 3] → pop 1
heap = [3, 5] → pop 2
heap = [5, 6] → pop 3
heap = [5, 6] → pop 4

最終 heap.top() = 5 （第 2 大）
