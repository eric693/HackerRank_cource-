#56 MEDIUM

Merge Intervals

Array Sorting

題目描述

以陣列 intervals 表示若干個區間的集合，其中單個區間為 intervals[i] = [starti, endi]。請你合併所有重疊的區間，並返回一個不重疊的區間陣列，該陣列需恰好覆蓋輸入中的所有區間。

範例

範例 1:

輸入：intervals = [[1,3],[2,6],[8,10],[15,18]]
輸出：[[1,6],[8,10],[15,18]]
解釋：區間 [1,3] 和 [2,6] 重疊，將它們合併為 [1,6]。

範例 2:

輸入：intervals = [[1,4],[4,5]]
輸出：[[1,5]]
解釋：區間 [1,4] 和 [4,5] 可被視為重疊區間。

限制條件

  • 1 <= intervals.length <= 104
  • intervals[i].length == 2
  • 0 <= starti <= endi <= 104

解題思路

排序 + 合併（推薦）

關鍵：先排序，然後逐個檢查是否重疊。

  1. 按照區間的起始位置排序
  2. 遍歷排序後的區間
  3. 如果當前區間與上一個區間重疊，合併它們
  4. 否則，將當前區間加入結果

重疊判斷：當前區間的起始 ≤ 上一個區間的結束

  • 時間複雜度：O(n log n) - 排序
  • 空間複雜度：O(log n) - 排序的堆疊空間

程式碼實作

Python 解法

class Solution:
    def merge(self, intervals: List[List[int]]) -> List[List[int]]:
        """
        排序 + 合併
        
        時間複雜度：O(n log n)
        空間複雜度：O(log n)
        """
        if not intervals:
            return []
        
        # 按照起始位置排序
        intervals.sort(key=lambda x: x[0])
        
        merged = [intervals[0]]
        
        for current in intervals[1:]:
            last = merged[-1]
            
            # 如果當前區間與上一個區間重疊
            if current[0] <= last[1]:
                # 合併區間：更新結束位置為兩者的最大值
                last[1] = max(last[1], current[1])
            else:
                # 不重疊，直接加入結果
                merged.append(current)
        
        return merged


# 方法二：更直觀的寫法
class Solution2:
    def merge(self, intervals: List[List[int]]) -> List[List[int]]:
        """
        更易理解的版本
        """
        intervals.sort()
        result = []
        
        for interval in intervals:
            # 如果結果為空或當前區間不重疊
            if not result or result[-1][1] < interval[0]:
                result.append(interval)
            else:
                # 重疊，合併區間
                result[-1][1] = max(result[-1][1], interval[1])
        
        return result

C++ 解法

#include <vector>
#include <algorithm>
using namespace std;

class Solution {
public:
    vector<vector<int>> merge(vector<vector<int>>& intervals) {
        if (intervals.empty()) return {};
        
        // 按照起始位置排序
        sort(intervals.begin(), intervals.end());
        
        vector<vector<int>> merged;
        merged.push_back(intervals[0]);
        
        for (int i = 1; i < intervals.size(); i++) {
            vector<int>& last = merged.back();
            vector<int>& current = intervals[i];
            
            // 如果重疊
            if (current[0] <= last[1]) {
                last[1] = max(last[1], current[1]);
            } else {
                merged.push_back(current);
            }
        }
        
        return merged;
    }
};

Java 解法

import java.util.*;

class Solution {
    public int[][] merge(int[][] intervals) {
        if (intervals.length == 0) return new int[0][];
        
        // 按照起始位置排序
        Arrays.sort(intervals, (a, b) -> Integer.compare(a[0], b[0]));
        
        List<int[]> merged = new ArrayList<>();
        merged.add(intervals[0]);
        
        for (int i = 1; i < intervals.length; i++) {
            int[] last = merged.get(merged.size() - 1);
            int[] current = intervals[i];
            
            // 如果重疊
            if (current[0] <= last[1]) {
                last[1] = Math.max(last[1], current[1]);
            } else {
                merged.add(current);
            }
        }
        
        return merged.toArray(new int[merged.size()][]);
    }
}

面試要點

常見追問

  • 如果插入一個新區間？（Insert Interval）
  • 如何處理開區間和閉區間？
  • 能否不排序就解決？
  • 如何找出重疊區間的數量？

解題技巧

  • 排序是解決區間問題的常見技巧
  • 重疊判斷：start ≤ last_end
  • 合併時更新為最大的結束位置
  • 注意邊界情況（相鄰但不重疊）

常見錯誤

  • 忘記排序
  • 重疊判斷條件錯誤
  • 合併時沒有取最大的結束位置
  • 沒有處理空陣列

視覺化說明

以 [[1,3],[2,6],[8,10],[15,18]] 為例：

排序後：
[1,3]  [2,6]  [8,10]  [15,18]
|--|    |---|   |--|     |---|
  
處理過程：
1. [1,3] → 加入結果
2. [2,6] → 與 [1,3] 重疊 → 合併為 [1,6]
3. [8,10] → 不重疊 → 加入結果
4. [15,18] → 不重疊 → 加入結果

最終結果：
[1,6]  [8,10]  [15,18]
|-----|  |--|    |---|
