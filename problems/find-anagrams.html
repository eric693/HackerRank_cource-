#438 MEDIUM

Find All Anagrams in a String

Hash Table String Sliding Window

題目描述

給定兩個字串 s 和 p，找到 s 中所有 p 的字母異位詞的子串，返回這些子串的起始索引。不考慮答案輸出的順序。

字母異位詞指由相同字母重排列形成的字串（包括相同的字串）。

範例

範例 1:

輸入：s = "cbaebabacd", p = "abc"
輸出：[0,6]
解釋：
起始索引等於 0 的子串是 "cba"，它是 "abc" 的字母異位詞。
起始索引等於 6 的子串是 "bac"，它是 "abc" 的字母異位詞。

範例 2:

輸入：s = "abab", p = "ab"
輸出：[0,1,2]
解釋：
起始索引等於 0 的子串是 "ab"，它是 "ab" 的字母異位詞。
起始索引等於 1 的子串是 "ba"，它是 "ab" 的字母異位詞。
起始索引等於 2 的子串是 "ab"，它是 "ab" 的字母異位詞。

限制條件

  • 1 <= s.length, p.length <= 3 * 104
  • s 和 p 僅包含小寫英文字母

解題思路

核心思想

使用滑動視窗 + 字符計數來高效檢查異位詞。

方法一：滑動視窗 + 字符計數（推薦）

  1. 統計 p 的字符頻率
  2. 使用固定大小的滑動視窗（長度 = len(p)）
  3. 維護視窗內的字符頻率
  4. 比較視窗頻率與 p 的頻率
  • 時間複雜度：O(n) - n 為 s 的長度
  • 空間複雜度：O(1) - 最多 26 個字母

優化技巧

  • 用計數器而非每次比較陣列
  • 用變量 matches 記錄匹配的字符數
  • 只需維護視窗邊界的變化

程式碼實作

Python 解法

from collections import Counter

class Solution:
    def findAnagrams(self, s: str, p: str) -> List[int]:
        """
        滑動視窗 + 字符計數
        
        時間複雜度：O(n)
        空間複雜度：O(1)
        """
        if len(p) > len(s):
            return []
        
        result = []
        p_count = Counter(p)
        window_count = Counter()
        
        # 初始化視窗
        for i in range(len(p)):
            window_count[s[i]] += 1
        
        # 檢查第一個視窗
        if window_count == p_count:
            result.append(0)
        
        # 滑動視窗
        for i in range(len(p), len(s)):
            # 加入新字符
            window_count[s[i]] += 1
            
            # 移除舊字符
            left_char = s[i - len(p)]
            window_count[left_char] -= 1
            if window_count[left_char] == 0:
                del window_count[left_char]
            
            # 檢查是否為異位詞
            if window_count == p_count:
                result.append(i - len(p) + 1)
        
        return result


# 方法二：優化版 - 使用 matches 計數
class Solution2:
    def findAnagrams(self, s: str, p: str) -> List[int]:
        """
        優化版滑動視窗
        
        時間複雜度：O(n)
        空間複雜度：O(1)
        """
        if len(p) > len(s):
            return []
        
        result = []
        p_count = [0] * 26
        s_count = [0] * 26
        
        # 統計 p 的字符頻率
        for char in p:
            p_count[ord(char) - ord('a')] += 1
        
        # 初始化視窗
        for i in range(len(p)):
            s_count[ord(s[i]) - ord('a')] += 1
        
        # 計算初始匹配數
        matches = sum(1 for i in range(26) if p_count[i] == s_count[i])
        
        # 檢查第一個視窗
        if matches == 26:
            result.append(0)
        
        # 滑動視窗
        for i in range(len(p), len(s)):
            # 加入新字符
            index = ord(s[i]) - ord('a')
            s_count[index] += 1
            if s_count[index] == p_count[index]:
                matches += 1
            elif s_count[index] == p_count[index] + 1:
                matches -= 1
            
            # 移除舊字符
            index = ord(s[i - len(p)]) - ord('a')
            s_count[index] -= 1
            if s_count[index] == p_count[index]:
                matches += 1
            elif s_count[index] == p_count[index] - 1:
                matches -= 1
            
            # 檢查是否匹配
            if matches == 26:
                result.append(i - len(p) + 1)
        
        return result

C++ 解法

#include <vector>
#include <string>
using namespace std;

class Solution {
public:
    vector<int> findAnagrams(string s, string p) {
        vector<int> result;
        if (p.length() > s.length()) return result;
        
        vector<int> pCount(26, 0), sCount(26, 0);
        
        // 統計 p 的字符頻率
        for (char c : p) {
            pCount[c - 'a']++;
        }
        
        // 初始化視窗
        for (int i = 0; i < p.length(); i++) {
            sCount[s[i] - 'a']++;
        }
        
        // 檢查第一個視窗
        if (pCount == sCount) {
            result.push_back(0);
        }
        
        // 滑動視窗
        for (int i = p.length(); i < s.length(); i++) {
            sCount[s[i] - 'a']++;
            sCount[s[i - p.length()] - 'a']--;
            
            if (pCount == sCount) {
                result.push_back(i - p.length() + 1);
            }
        }
        
        return result;
    }
};

Java 解法

import java.util.*;

class Solution {
    public List<Integer> findAnagrams(String s, String p) {
        List<Integer> result = new ArrayList<>();
        if (p.length() > s.length()) return result;
        
        int[] pCount = new int[26];
        int[] sCount = new int[26];
        
        // 統計 p 的字符頻率
        for (char c : p.toCharArray()) {
            pCount[c - 'a']++;
        }
        
        // 初始化視窗
        for (int i = 0; i < p.length(); i++) {
            sCount[s.charAt(i) - 'a']++;
        }
        
        // 檢查第一個視窗
        if (Arrays.equals(pCount, sCount)) {
            result.add(0);
        }
        
        // 滑動視窗
        for (int i = p.length(); i < s.length(); i++) {
            sCount[s.charAt(i) - 'a']++;
            sCount[s.charAt(i - p.length()) - 'a']--;
            
            if (Arrays.equals(pCount, sCount)) {
                result.add(i - p.length() + 1);
            }
        }
        
        return result;
    }
}

面試要點

常見追問

  • 如何優化到 O(n)？
  • 能否不使用額外空間？
  • 如何處理大小寫？
  • matches 變量的作用？

解題技巧

  • 固定大小的滑動視窗
  • 字符計數比較
  • matches 優化避免每次比較陣列
  • 注意視窗邊界處理

常見錯誤

  • 視窗大小計算錯誤
  • 起始索引計算錯誤
  • 字符計數更新遺漏
  • 忘記檢查第一個視窗

滑動視窗視覺化

以 s = "cbaebabacd", p = "abc" 為例：

p = "abc" → 計數: {a:1, b:1, c:1}

視窗大小 = 3

步驟 1: [c b a] e b a b a c d
        計數: {c:1, b:1, a:1} ✓ 匹配！索引 0

步驟 2: c [b a e] b a b a c d
        計數: {b:1, a:1, e:1} ✗

步驟 3: c b [a e b] a b a c d
        計數: {a:1, e:1, b:1} ✗

步驟 4: c b a [e b a] b a c d
        計數: {e:1, b:1, a:1} ✗

步驟 5: c b a e [b a b] a c d
        計數: {b:2, a:1} ✗

步驟 6: c b a e b [a b a] c d
        計數: {a:2, b:1} ✗

步驟 7: c b a e b a [b a c] d
        計數: {b:1, a:1, c:1} ✓ 匹配！索引 6

結果：[0, 6]

matches 優化說明

使用 matches 變量避免每次比較整個陣列：

  • matches：記錄有多少個字符頻率匹配（共 26 個字母）
  • 當 matches == 26 時，視窗內容是異位詞
  • 只需維護邊界字符的 matches 變化

時間優化：從 O(26n) 降到 O(n)
