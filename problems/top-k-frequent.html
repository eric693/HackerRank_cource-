#347 MEDIUM

Top K Frequent Elements

Array Hash Table Heap Bucket Sort

題目描述

給你一個整數陣列 nums 和一個整數 k，請你返回其中出現頻率前 k 高的元素。你可以按任意順序返回答案。

範例

範例 1:

輸入：nums = [1,1,1,2,2,3], k = 2
輸出：[1,2]

範例 2:

輸入：nums = [1], k = 1
輸出：[1]

限制條件

  • 1 <= nums.length <= 105
  • -104 <= nums[i] <= 104
  • k 的取值範圍是 [1, 陣列中不相同的元素的個數]
  • 題目資料保證答案唯一，換句話說，陣列中前 k 個高頻元素的集合是唯一的

解題思路

方法一：桶排序（推薦）- O(n)

利用頻率作為桶的索引：

  1. 統計每個元素的出現頻率
  2. 創建大小為 n+1 的桶陣列（頻率最高為 n）
  3. 將元素按頻率放入對應的桶中
  4. 從高頻到低頻收集前 k 個元素
  • 時間複雜度：O(n)
  • 空間複雜度：O(n)

方法二：最小堆 - O(n log k)

維護大小為 k 的最小堆：

  1. 統計每個元素的出現頻率
  2. 維護一個大小為 k 的最小堆
  3. 遍歷所有元素，將頻率較高的保留在堆中
  • 時間複雜度：O(n log k)
  • 空間複雜度：O(n)

方法三：快速選擇 - O(n) 平均

類似快速排序的分區思想。

程式碼實作

Python 解法

from collections import Counter
from typing import List
import heapq

class Solution:
    def topKFrequent(self, nums: List[int], k: int) -> List[int]:
        """
        方法一：桶排序 O(n)
        
        時間複雜度：O(n)
        空間複雜度：O(n)
        """
        # 統計頻率
        count = Counter(nums)
        
        # 創建桶（頻率作為索引）
        buckets = [[] for _ in range(len(nums) + 1)]
        
        # 將元素放入對應頻率的桶中
        for num, freq in count.items():
            buckets[freq].append(num)
        
        # 從高頻到低頻收集結果
        result = []
        for i in range(len(buckets) - 1, 0, -1):
            result.extend(buckets[i])
            if len(result) >= k:
                return result[:k]
        
        return result


class Solution2:
    def topKFrequent(self, nums: List[int], k: int) -> List[int]:
        """
        方法二：最小堆 O(n log k)
        
        時間複雜度：O(n log k)
        空間複雜度：O(n)
        """
        count = Counter(nums)
        
        # 使用 heapq.nlargest 獲取頻率最高的 k 個元素
        return heapq.nlargest(k, count.keys(), key=count.get)


class Solution3:
    def topKFrequent(self, nums: List[int], k: int) -> List[int]:
        """
        方法三：手動維護最小堆
        
        時間複雜度：O(n log k)
        空間複雜度：O(n)
        """
        count = Counter(nums)
        
        # 維護大小為 k 的最小堆
        heap = []
        for num, freq in count.items():
            heapq.heappush(heap, (freq, num))
            if len(heap) > k:
                heapq.heappop(heap)
        
        return [num for freq, num in heap]

C++ 解法

#include <vector>
#include <unordered_map>
#include <queue>
using namespace std;

class Solution {
public:
    vector<int> topKFrequent(vector<int>& nums, int k) {
        // 統計頻率
        unordered_map<int, int> count;
        for (int num : nums) {
            count[num]++;
        }
        
        // 創建桶
        vector<vector<int>> buckets(nums.size() + 1);
        for (auto& p : count) {
            buckets[p.second].push_back(p.first);
        }
        
        // 從高頻到低頻收集結果
        vector<int> result;
        for (int i = buckets.size() - 1; i >= 0 && result.size() < k; i--) {
            result.insert(result.end(), buckets[i].begin(), buckets[i].end());
        }
        
        result.resize(k);
        return result;
    }
};

// 方法二：使用優先隊列（最小堆）
class Solution2 {
public:
    vector<int> topKFrequent(vector<int>& nums, int k) {
        unordered_map<int, int> count;
        for (int num : nums) {
            count[num]++;
        }
        
        // 最小堆：pair<頻率, 數字>
        priority_queue<pair<int,int>, vector<pair<int,int>>, greater<pair<int,int>>> minHeap;
        
        for (auto& p : count) {
            minHeap.push({p.second, p.first});
            if (minHeap.size() > k) {
                minHeap.pop();
            }
        }
        
        vector<int> result;
        while (!minHeap.empty()) {
            result.push_back(minHeap.top().second);
            minHeap.pop();
        }
        
        return result;
    }
};

Java 解法

import java.util.*;

class Solution {
    public int[] topKFrequent(int[] nums, int k) {
        // 統計頻率
        Map<Integer, Integer> count = new HashMap<>();
        for (int num : nums) {
            count.put(num, count.getOrDefault(num, 0) + 1);
        }
        
        // 創建桶
        List<Integer>[] buckets = new List[nums.length + 1];
        for (int i = 0; i < buckets.length; i++) {
            buckets[i] = new ArrayList<>();
        }
        
        for (Map.Entry<Integer, Integer> entry : count.entrySet()) {
            buckets[entry.getValue()].add(entry.getKey());
        }
        
        // 從高頻到低頻收集結果
        int[] result = new int[k];
        int index = 0;
        for (int i = buckets.length - 1; i >= 0 && index < k; i--) {
            for (int num : buckets[i]) {
                result[index++] = num;
                if (index == k) break;
            }
        }
        
        return result;
    }
}

步驟 2：創建桶（頻率作為索引）
buckets = [
  [],      # 頻率 0
  [3],     # 頻率 1
  [2],     # 頻率 2
  [1],     # 頻率 3
  [],      # 頻率 4
  [],      # 頻率 5
  []       # 頻率 6
]

步驟 3：從高頻到低頻收集
i = 6: buckets[6] = [] → result = []
i = 5: buckets[5] = [] → result = []
i = 4: buckets[4] = [] → result = []
i = 3: buckets[3] = [1] → result = [1]

已收集 k=2 個元素，返回 [1, 2]

面試要點

常見追問

  • 為什麼桶排序是 O(n)？
  • 如果 k 很小，哪種方法更好？
  • 如何處理頻率相同的元素？
  • 能否在 O(n log n) 內解決？

解題技巧

  • 桶排序適合頻率範圍有限
  • 最小堆適合 k 很小的情況
  • Counter 可快速統計頻率
  • 理解不同方法的權衡

常見錯誤

  • 桶的大小設置錯誤
  • 忘記處理空桶
  • 收集結果時超出 k 個
  • 堆的大小控制不當
