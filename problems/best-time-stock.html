#121 EASY

Best Time to Buy and Sell Stock

Array Dynamic Programming

題目描述

給定一個陣列 prices，其中 prices[i] 是第 i 天某支股票的價格。

你想要透過選擇某一天買入這支股票，並選擇在未來的某一個不同的日子賣出該股票，以最大化你的利潤。

返回你可以從這筆交易中獲取的最大利潤。如果你不能獲取任何利潤，返回 0。

範例

範例 1:

輸入：prices = [7,1,5,3,6,4]
輸出：5
解釋：在第 2 天（股票價格 = 1）買入，在第 5 天（股票價格 = 6）賣出，
     最大利潤 = 6 - 1 = 5。
     注意：利潤不能是 7 - 1 = 6，因為賣出價格需要大於買入價格；
     同時，你不能在買入前賣出股票。

範例 2:

輸入：prices = [7,6,4,3,1]
輸出：0
解釋：在這種情況下，沒有交易完成，所以最大利潤為 0。

限制條件

  • 1 <= prices.length <= 105
  • 0 <= prices[i] <= 104

解題思路

一次遍歷（推薦）

關鍵觀察：要獲得最大利潤，需要找到「最低買入價」和「最高賣出價」的組合。

  1. 維護一個變數記錄目前遇到的最低價格
  2. 對每個價格，計算「今天賣出」能獲得的利潤
  3. 更新最大利潤
  • 時間複雜度：O(n) - 只需遍歷一次
  • 空間複雜度：O(1)

暴力法（不推薦）

檢查所有可能的買入賣出組合。

  • 時間複雜度：O(n²) - 會超時
  • 空間複雜度：O(1)

程式碼實作

Python 解法

class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        """
        一次遍歷解法
        
        時間複雜度：O(n)
        空間複雜度：O(1)
        """
        if not prices:
            return 0
        
        min_price = float('inf')
        max_profit = 0
        
        for price in prices:
            # 更新最低價格
            min_price = min(min_price, price)
            
            # 計算今天賣出的利潤
            profit = price - min_price
            
            # 更新最大利潤
            max_profit = max(max_profit, profit)
        
        return max_profit


# 方法二：更簡潔的寫法
class Solution2:
    def maxProfit(self, prices: List[int]) -> int:
        """
        一行式寫法（不推薦，可讀性差）
        """
        min_price = float('inf')
        max_profit = 0
        
        for price in prices:
            min_price = min(min_price, price)
            max_profit = max(max_profit, price - min_price)
        
        return max_profit


# 方法三：動態規劃思維
class Solution3:
    def maxProfit(self, prices: List[int]) -> int:
        """
        動態規劃解法
        
        狀態定義：
        - dp[i][0]: 第 i 天不持有股票的最大利潤
        - dp[i][1]: 第 i 天持有股票的最大利潤
        """
        if not prices:
            return 0
        
        # 初始狀態
        hold = -prices[0]  # 持有股票
        not_hold = 0       # 不持有股票
        
        for i in range(1, len(prices)):
            # 今天不持有 = max(昨天不持有, 昨天持有今天賣)
            not_hold = max(not_hold, hold + prices[i])
            
            # 今天持有 = max(昨天持有, 今天買入)
            # 因為只能交易一次，所以買入成本就是 -prices[i]
            hold = max(hold, -prices[i])
        
        return not_hold

C++ 解法

#include <vector>
#include <algorithm>
#include <climits>
using namespace std;

class Solution {
public:
    int maxProfit(vector<int>& prices) {
        if (prices.empty()) return 0;
        
        int minPrice = INT_MAX;
        int maxProfit = 0;
        
        for (int price : prices) {
            // 更新最低價格
            minPrice = min(minPrice, price);
            
            // 計算今天賣出的利潤
            int profit = price - minPrice;
            
            // 更新最大利潤
            maxProfit = max(maxProfit, profit);
        }
        
        return maxProfit;
    }
};

Java 解法

class Solution {
    public int maxProfit(int[] prices) {
        if (prices.length == 0) return 0;
        
        int minPrice = Integer.MAX_VALUE;
        int maxProfit = 0;
        
        for (int price : prices) {
            // 更新最低價格
            minPrice = Math.min(minPrice, price);
            
            // 計算今天賣出的利潤
            int profit = price - minPrice;
            
            // 更新最大利潤
            maxProfit = Math.max(maxProfit, profit);
        }
        
        return maxProfit;
    }
}

面試要點

常見追問

  • 如果可以多次買賣怎麼辦？（Best Time II）
  • 如果最多只能交易 k 次？（Best Time IV）
  • 如果有手續費？（Best Time with Fee）
  • 如何返回買賣的具體日期？

解題技巧

  • 追蹤最低買入價是關鍵
  • 對每個價格計算「今天賣出」的利潤
  • 一次遍歷即可解決
  • 這是 DP 系列題的入門

常見錯誤

  • 使用 O(n²) 暴力法導致超時
  • 忘記檢查空陣列
  • 沒有正確更新最低價格
  • 誤解題意，以為可以多次交易

延伸問題

Best Time to Buy and Sell Stock 系列

  1. I（本題）：只能交易一次
  2. II：可以多次交易（無限次）
  3. III：最多交易兩次
  4. IV：最多交易 k 次
  5. with Cooldown：賣出後需要冷卻一天
  6. with Transaction Fee：每次交易需要手續費

這是一個經典的動態規劃系列題，由易到難，非常適合學習狀態機 DP。
