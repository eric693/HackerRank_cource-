#994 MEDIUM

Rotting Oranges

Array BFS Matrix

題目描述

在給定的 m x n 網格 grid 中，每個單元格可以有以下三個值之一：

  • 值 0 代表空單元格
  • 值 1 代表新鮮橘子
  • 值 2 代表腐爛的橘子

每分鐘，腐爛的橘子周圍四個方向上相鄰的新鮮橘子都會腐爛。

返回直到單元格中沒有新鮮橘子為止所必須經過的最小分鐘數。如果不可能，返回 -1。

範例

範例 1:

輸入：grid = [[2,1,1],[1,1,0],[0,1,1]]
輸出：4

過程：
分鐘 0: [[2,1,1],    分鐘 1: [[2,2,1],
        [1,1,0],            [2,1,0],
        [0,1,1]]            [0,1,1]]

分鐘 2: [[2,2,2],    分鐘 3: [[2,2,2],
        [2,2,0],            [2,2,0],
        [0,1,1]]            [0,2,1]]

分鐘 4: [[2,2,2],
        [2,2,0],
        [0,2,2]]

範例 2:

輸入：grid = [[2,1,1],[0,1,1],[1,0,1]]
輸出：-1
解釋：左下角的橘子（第 2 行，第 0 列）永遠不會腐爛，因為腐爛只會發生在 4 個方向上。

範例 3:

輸入：grid = [[0,2]]
輸出：0
解釋：因為 0 分鐘時已經沒有新鮮橘子了，所以答案就是 0。

限制條件

  • m == grid.length
  • n == grid[i].length
  • 1 <= m, n <= 10
  • grid[i][j] 僅為 0、1 或 2

解題思路

多源 BFS（推薦）

這是一個經典的多源 BFS 問題：

  1. 將所有腐爛的橘子加入佇列作為起點
  2. 統計新鮮橘子的數量
  3. BFS 擴散：每分鐘處理當前層的所有腐爛橘子
  4. 腐爛相鄰的新鮮橘子，並加入佇列
  5. 最後檢查是否還有新鮮橘子

關鍵點

  • 多源 BFS：同時從多個起點開始
  • 層序遍歷：每層代表一分鐘
  • 四個方向擴散
  • 時間複雜度：O(m × n)
  • 空間複雜度：O(m × n)

程式碼實作

Python 解法

from collections import deque

class Solution:
    def orangesRotting(self, grid: List[List[int]]) -> int:
        """
        多源 BFS
        
        時間複雜度：O(m × n)
        空間複雜度：O(m × n)
        """
        rows, cols = len(grid), len(grid[0])
        queue = deque()
        fresh_count = 0
        
        # 統計新鮮橘子數量，並將腐爛橘子加入佇列
        for r in range(rows):
            for c in range(cols):
                if grid[r][c] == 2:
                    queue.append((r, c))
                elif grid[r][c] == 1:
                    fresh_count += 1
        
        # 如果沒有新鮮橘子，直接返回 0
        if fresh_count == 0:
            return 0
        
        # 四個方向
        directions = [(0, 1), (0, -1), (1, 0), (-1, 0)]
        minutes = 0
        
        # BFS
        while queue:
            # 處理當前層（一分鐘內的所有橘子）
            size = len(queue)
            
            for _ in range(size):
                r, c = queue.popleft()
                
                # 向四個方向擴散
                for dr, dc in directions:
                    nr, nc = r + dr, c + dc
                    
                    # 檢查邊界和是否為新鮮橘子
                    if (0 <= nr < rows and 0 <= nc < cols and 
                        grid[nr][nc] == 1):
                        # 腐爛這個橘子
                        grid[nr][nc] = 2
                        fresh_count -= 1
                        queue.append((nr, nc))
            
            # 如果佇列還有元素，增加時間
            if queue:
                minutes += 1
        
        # 檢查是否還有新鮮橘子
        return minutes if fresh_count == 0 else -1

C++ 解法

#include <vector>
#include <queue>
using namespace std;

class Solution {
public:
    int orangesRotting(vector<vector<int>>& grid) {
        int rows = grid.size();
        int cols = grid[0].size();
        queue<pair<int, int>> q;
        int freshCount = 0;
        
        // 統計新鮮橘子並收集腐爛橘子
        for (int r = 0; r < rows; r++) {
            for (int c = 0; c < cols; c++) {
                if (grid[r][c] == 2) {
                    q.push({r, c});
                } else if (grid[r][c] == 1) {
                    freshCount++;
                }
            }
        }
        
        if (freshCount == 0) return 0;
        
        int directions[4][2] = {{0,1}, {0,-1}, {1,0}, {-1,0}};
        int minutes = 0;
        
        while (!q.empty()) {
            int size = q.size();
            
            for (int i = 0; i < size; i++) {
                auto [r, c] = q.front();
                q.pop();
                
                for (auto& dir : directions) {
                    int nr = r + dir[0];
                    int nc = c + dir[1];
                    
                    if (nr >= 0 && nr < rows && nc >= 0 && nc < cols && 
                        grid[nr][nc] == 1) {
                        grid[nr][nc] = 2;
                        freshCount--;
                        q.push({nr, nc});
                    }
                }
            }
            
            if (!q.empty()) minutes++;
        }
        
        return freshCount == 0 ? minutes : -1;
    }
};

Java 解法

import java.util.*;

class Solution {
    public int orangesRotting(int[][] grid) {
        int rows = grid.length;
        int cols = grid[0].length;
        Queue<int[]> queue = new LinkedList<>();
        int freshCount = 0;
        
        // 統計新鮮橘子並收集腐爛橘子
        for (int r = 0; r < rows; r++) {
            for (int c = 0; c < cols; c++) {
                if (grid[r][c] == 2) {
                    queue.offer(new int[]{r, c});
                } else if (grid[r][c] == 1) {
                    freshCount++;
                }
            }
        }
        
        if (freshCount == 0) return 0;
        
        int[][] directions = {{0,1}, {0,-1}, {1,0}, {-1,0}};
        int minutes = 0;
        
        while (!queue.isEmpty()) {
            int size = queue.size();
            
            for (int i = 0; i < size; i++) {
                int[] pos = queue.poll();
                int r = pos[0], c = pos[1];
                
                for (int[] dir : directions) {
                    int nr = r + dir[0];
                    int nc = c + dir[1];
                    
                    if (nr >= 0 && nr < rows && nc >= 0 && nc < cols && 
                        grid[nr][nc] == 1) {
                        grid[nr][nc] = 2;
                        freshCount--;
                        queue.offer(new int[]{nr, nc});
                    }
                }
            }
            
            if (!queue.isEmpty()) minutes++;
        }
        
        return freshCount == 0 ? minutes : -1;
    }
}

為什麼用 BFS？

BFS 適合處理「層級」或「時間步」的問題：

  • 層序遍歷：每層代表一個時間單位
  • 最短路徑：BFS 保證找到最短時間
  • 同時擴散：多個腐爛橘子同時向外擴散

DFS 無法保證找到最短時間，因為它會一條路走到底。
