#5 MEDIUM

Longest Palindromic Substring

String Dynamic Programming

題目描述

給定一個字串 s，找到 s 中最長的回文子字串。

回文是正著讀和反著讀都一樣的字串。

範例

範例 1:

輸入：s = "babad"
輸出："bab"
解釋："aba" 同樣是符合題意的答案。

範例 2:

輸入：s = "cbbd"
輸出："bb"

限制條件

  • 1 <= s.length <= 1000
  • s 僅由數字和英文字母組成

解題思路

方法一：中心擴展法（推薦）

從每個可能的中心點向兩邊擴展：

  • 奇數長度回文：中心是一個字符
  • 偶數長度回文：中心是兩個字符之間
  • 時間複雜度：O(n²)
  • 空間複雜度：O(1)

方法二：動態規劃

定義 dp[i][j] 表示 s[i:j+1] 是否為回文。

  • 時間複雜度：O(n²)
  • 空間複雜度：O(n²)

方法三：Manacher's Algorithm（最優）

線性時間算法，但實現較複雜。

  • 時間複雜度：O(n)
  • 空間複雜度：O(n)

程式碼實作

Python 解法

class Solution:
    def longestPalindrome(self, s: str) -> str:
        """
        中心擴展法
        
        時間複雜度：O(n²)
        空間複雜度：O(1)
        """
        if not s:
            return ""
        
        start = 0
        max_len = 0
        
        def expand_around_center(left: int, right: int) -> int:
            """從中心向兩邊擴展，返回回文長度"""
            while left >= 0 and right < len(s) and s[left] == s[right]:
                left -= 1
                right += 1
            return right - left - 1
        
        for i in range(len(s)):
            # 奇數長度回文（中心是一個字符）
            len1 = expand_around_center(i, i)
            # 偶數長度回文（中心是兩個字符之間）
            len2 = expand_around_center(i, i + 1)
            
            # 取較長的回文
            length = max(len1, len2)
            
            if length > max_len:
                max_len = length
                # 計算起始位置
                start = i - (length - 1) // 2
        
        return s[start:start + max_len]


# 方法二：動態規劃
class Solution2:
    def longestPalindrome(self, s: str) -> str:
        """
        動態規劃
        
        時間複雜度：O(n²)
        空間複雜度：O(n²)
        """
        n = len(s)
        if n < 2:
            return s
        
        # dp[i][j] 表示 s[i:j+1] 是否為回文
        dp = [[False] * n for _ in range(n)]
        
        start = 0
        max_len = 1
        
        # 單個字符都是回文
        for i in range(n):
            dp[i][i] = True
        
        # 填表，按照子字串長度遞增
        for length in range(2, n + 1):
            for i in range(n - length + 1):
                j = i + length - 1
                
                if s[i] == s[j]:
                    if length == 2:
                        dp[i][j] = True
                    else:
                        dp[i][j] = dp[i + 1][j - 1]
                
                if dp[i][j] and length > max_len:
                    start = i
                    max_len = length
        
        return s[start:start + max_len]


# 方法三：更簡潔的中心擴展
class Solution3:
    def longestPalindrome(self, s: str) -> str:
        """
        簡潔版中心擴展
        """
        def expand(left, right):
            while left >= 0 and right < len(s) and s[left] == s[right]:
                left -= 1
                right += 1
            return s[left + 1:right]
        
        result = ""
        for i in range(len(s)):
            # 奇數和偶數長度
            palindrome = max(expand(i, i), expand(i, i + 1), key=len)
            if len(palindrome) > len(result):
                result = palindrome
        
        return result

C++ 解法

#include <string>
using namespace std;

class Solution {
public:
    string longestPalindrome(string s) {
        if (s.empty()) return "";
        
        int start = 0, maxLen = 0;
        
        for (int i = 0; i < s.length(); i++) {
            // 奇數長度
            int len1 = expandAroundCenter(s, i, i);
            // 偶數長度
            int len2 = expandAroundCenter(s, i, i + 1);
            
            int len = max(len1, len2);
            
            if (len > maxLen) {
                maxLen = len;
                start = i - (len - 1) / 2;
            }
        }
        
        return s.substr(start, maxLen);
    }
    
private:
    int expandAroundCenter(const string& s, int left, int right) {
        while (left >= 0 && right < s.length() && s[left] == s[right]) {
            left--;
            right++;
        }
        return right - left - 1;
    }
};

Java 解法

class Solution {
    public String longestPalindrome(String s) {
        if (s == null || s.length() == 0) return "";
        
        int start = 0, maxLen = 0;
        
        for (int i = 0; i < s.length(); i++) {
            // 奇數長度
            int len1 = expandAroundCenter(s, i, i);
            // 偶數長度
            int len2 = expandAroundCenter(s, i, i + 1);
            
            int len = Math.max(len1, len2);
            
            if (len > maxLen) {
                maxLen = len;
                start = i - (len - 1) / 2;
            }
        }
        
        return s.substring(start, start + maxLen);
    }
    
    private int expandAroundCenter(String s, int left, int right) {
        while (left >= 0 && right < s.length() && s.charAt(left) == s.charAt(right)) {
            left--;
            right++;
        }
        return right - left - 1;
    }
}

面試要點

常見追問

  • 如何判斷整個字串是否為回文？
  • 能否在 O(n) 時間內解決？
  • 如何找出所有回文子字串？
  • 最長回文子序列有什麼不同？

解題技巧

  • 中心擴展法最直觀且高效
  • 注意奇偶長度兩種情況
  • DP 方法空間換時間
  • Manacher 算法最優但複雜

常見錯誤

  • 忘記處理偶數長度回文
  • 起始位置計算錯誤
  • 邊界條件檢查遺漏
  • 只返回長度而非字串本身

中心擴展視覺化

以 "babad" 為例：

索引:  0 1 2 3 4
字串:  b a b a d

中心 i=0 (b):
  奇數: b (長度 1)
  偶數: (長度 0)

中心 i=1 (a):
  奇數: bab (長度 3) ← 找到回文
  偶數: (長度 0)

中心 i=2 (b):
  奇數: aba (長度 3) ← 也是回文
  偶數: (長度 0)

中心 i=3 (a):
  奇數: a (長度 1)
  偶數: (長度 0)

最長回文: "bab" 或 "aba" (長度 3)
