#53 MEDIUM

Maximum Subarray

Array Dynamic Programming Divide and Conquer

題目描述

給定一個整數陣列 nums，找出一個具有最大和的連續子陣列（子陣列最少包含一個元素），返回其最大和。

子陣列是陣列中的一個連續部分。

範例

範例 1:

輸入：nums = [-2,1,-3,4,-1,2,1,-5,4]
輸出：6
解釋：連續子陣列 [4,-1,2,1] 的和最大，為 6。

範例 2:

輸入：nums = [1]
輸出：1
解釋：連續子陣列 [1] 的和最大，為 1。

範例 3:

輸入：nums = [5,4,-1,7,8]
輸出：23
解釋：連續子陣列 [5,4,-1,7,8] 的和最大，為 23。

限制條件

  • 1 <= nums.length <= 105
  • -104 <= nums[i] <= 104

解題思路

Kadane's Algorithm（最優解）

Kadane 算法是解決最大子陣列和問題的經典動態規劃方法：

  1. 維護兩個變數：當前子陣列和（current_sum）和最大子陣列和（max_sum）
  2. 遍歷陣列，對每個元素決定是加入當前子陣列還是重新開始
  3. 如果當前和變成負數，不如重新開始
  4. 每次更新最大和
  • 時間複雜度：O(n)
  • 空間複雜度：O(1)

分治法（Divide and Conquer）

將陣列分成兩半，最大子陣列和可能在：

  • 左半部分
  • 右半部分
  • 跨越中間點
  • 時間複雜度：O(n log n)
  • 空間複雜度：O(log n)

程式碼實作

Python 解法：Kadane's Algorithm

class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        """
        Kadane's Algorithm - 動態規劃解法
        
        時間複雜度：O(n)
        空間複雜度：O(1)
        """
        # 初始化
        current_sum = max_sum = nums[0]
        
        # 從第二個元素開始遍歷
        for i in range(1, len(nums)):
            # 決定是加入當前子陣列還是重新開始
            current_sum = max(nums[i], current_sum + nums[i])
            # 更新最大和
            max_sum = max(max_sum, current_sum)
        
        return max_sum


# 方法二：更直觀的寫法
class Solution2:
    def maxSubArray(self, nums: List[int]) -> int:
        current_sum = 0
        max_sum = float('-inf')
        
        for num in nums:
            # 如果當前和是負數，不如重新開始
            if current_sum < 0:
                current_sum = 0
            
            current_sum += num
            max_sum = max(max_sum, current_sum)
        
        return max_sum


# 方法三：分治法
class Solution3:
    def maxSubArray(self, nums: List[int]) -> int:
        def helper(left, right):
            # 基本情況
            if left == right:
                return nums[left]
            
            # 分割
            mid = (left + right) // 2
            
            # 左半部分的最大和
            left_sum = helper(left, mid)
            # 右半部分的最大和
            right_sum = helper(mid + 1, right)
            # 跨越中間的最大和
            cross_sum = self.maxCrossingSum(left, mid, right)
            
            # 返回三者中的最大值
            return max(left_sum, right_sum, cross_sum)
        
        def maxCrossingSum(self, left, mid, right):
            # 從中間向左的最大和
            left_sum = float('-inf')
            current_sum = 0
            for i in range(mid, left - 1, -1):
                current_sum += nums[i]
                left_sum = max(left_sum, current_sum)
            
            # 從中間向右的最大和
            right_sum = float('-inf')
            current_sum = 0
            for i in range(mid + 1, right + 1):
                current_sum += nums[i]
                right_sum = max(right_sum, current_sum)
            
            return left_sum + right_sum
        
        return helper(0, len(nums) - 1)

C++ 解法：Kadane's Algorithm

#include <vector>
#include <algorithm>
using namespace std;

class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        // 初始化
        int currentSum = nums[0];
        int maxSum = nums[0];
        
        // 從第二個元素開始遍歷
        for (int i = 1; i < nums.size(); i++) {
            // 決定是加入當前子陣列還是重新開始
            currentSum = max(nums[i], currentSum + nums[i]);
            // 更新最大和
            maxSum = max(maxSum, currentSum);
        }
        
        return maxSum;
    }
};

Java 解法：Kadane's Algorithm

class Solution {
    public int maxSubArray(int[] nums) {
        // 初始化
        int currentSum = nums[0];
        int maxSum = nums[0];
        
        // 從第二個元素開始遍歷
        for (int i = 1; i < nums.length; i++) {
            // 決定是加入當前子陣列還是重新開始
            currentSum = Math.max(nums[i], currentSum + nums[i]);
            // 更新最大和
            maxSum = Math.max(maxSum, currentSum);
        }
        
        return maxSum;
    }
}

面試要點

常見追問

  • 如果需要返回子陣列的起始和結束索引呢？
  • 如果陣列中所有數字都是負數怎麼辦？
  • 能否用分治法解決？時間複雜度是多少？
  • 如何處理環形陣列的情況？

解題技巧

  • Kadane's Algorithm 是最優解
  • 理解動態規劃的狀態轉移方程
  • 注意全是負數的邊界情況
  • 可以優化到 O(1) 空間複雜度

常見錯誤

  • 初始化錯誤（應該用第一個元素而不是 0）
  • 沒有處理全負數的情況
  • 忘記更新最大和
  • 混淆「子陣列」和「子序列」的概念
