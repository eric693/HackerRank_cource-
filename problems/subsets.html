#78 MEDIUM

Subsets

Array Backtracking Bit Manipulation

題目描述

給定一個整數陣列 nums，陣列中的元素互不相同。返回該陣列所有可能的子集（冪集）。

解集不能包含重複的子集。你可以按任意順序返回解集。

範例

範例 1:

輸入：nums = [1,2,3]
輸出：[[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]

範例 2:

輸入：nums = [0]
輸出：[[],[0]]

限制條件

  • 1 <= nums.length <= 10
  • -10 <= nums[i] <= 10
  • nums 中的所有元素互不相同

解題思路

核心觀察

對於 n 個元素，共有 2^n 個子集（每個元素選或不選）。

方法一：回溯法（推薦）

對每個元素，有兩個選擇：選或不選。

  1. 從第一個元素開始
  2. 選擇當前元素，遞迴處理剩餘元素
  3. 不選當前元素，遞迴處理剩餘元素
  4. 回溯
  • 時間複雜度：O(n × 2^n) - 有 2^n 個子集，每個需要 O(n) 複製
  • 空間複雜度：O(n) - 遞迴深度

方法二：迭代法

從空集開始，依次加入每個元素。

  • 時間複雜度：O(n × 2^n)
  • 空間複雜度：O(1)

方法三：位元操作

用二進制表示每個子集（1 表示選，0 表示不選）。

  • 時間複雜度：O(n × 2^n)
  • 空間複雜度：O(1)

程式碼實作

Python 解法

class Solution:
    def subsets(self, nums: List[int]) -> List[List[int]]:
        """
        回溯法
        
        時間複雜度：O(n × 2^n)
        空間複雜度：O(n)
        """
        result = []
        
        def backtrack(start, path):
            # 每個路徑都是一個有效的子集
            result.append(path[:])
            
            # 嘗試加入後續的每個元素
            for i in range(start, len(nums)):
                # 選擇當前元素
                path.append(nums[i])
                
                # 遞迴處理下一個位置
                backtrack(i + 1, path)
                
                # 回溯：不選當前元素
                path.pop()
        
        backtrack(0, [])
        return result


# 方法二：迭代法
class Solution2:
    def subsets(self, nums: List[int]) -> List[List[int]]:
        """
        迭代法 - 從空集開始逐個加入元素
        
        時間複雜度：O(n × 2^n)
        空間複雜度：O(1)
        """
        result = [[]]
        
        for num in nums:
            # 對每個現有子集，創建加入當前元素的新子集
            result += [subset + [num] for subset in result]
        
        return result


# 方法三：位元操作
class Solution3:
    def subsets(self, nums: List[int]) -> List[List[int]]:
        """
        位元操作 - 用二進制表示每個子集
        
        時間複雜度：O(n × 2^n)
        空間複雜度：O(1)
        """
        n = len(nums)
        result = []
        
        # 遍歷從 0 到 2^n - 1 的所有數字
        for i in range(2 ** n):
            subset = []
            
            # 檢查第 j 位是否為 1
            for j in range(n):
                if (i >> j) & 1:
                    subset.append(nums[j])
            
            result.append(subset)
        
        return result


# 方法四：更簡潔的迭代寫法
class Solution4:
    def subsets(self, nums: List[int]) -> List[List[int]]:
        """
        更簡潔的迭代法
        """
        result = [[]]
        
        for num in nums:
            new_subsets = []
            for subset in result:
                new_subsets.append(subset + [num])
            result.extend(new_subsets)
        
        return result

C++ 解法

#include <vector>
using namespace std;

class Solution {
public:
    vector<vector<int>> subsets(vector<int>& nums) {
        vector<vector<int>> result;
        vector<int> path;
        backtrack(nums, 0, path, result);
        return result;
    }
    
private:
    void backtrack(vector<int>& nums, int start, 
                   vector<int>& path, vector<vector<int>>& result) {
        // 每個路徑都是一個有效的子集
        result.push_back(path);
        
        // 嘗試加入後續的每個元素
        for (int i = start; i < nums.size(); i++) {
            // 選擇當前元素
            path.push_back(nums[i]);
            
            // 遞迴處理下一個位置
            backtrack(nums, i + 1, path, result);
            
            // 回溯
            path.pop_back();
        }
    }
};

Java 解法

import java.util.*;

class Solution {
    public List<List<Integer>> subsets(int[] nums) {
        List<List<Integer>> result = new ArrayList<>();
        backtrack(nums, 0, new ArrayList<>(), result);
        return result;
    }
    
    private void backtrack(int[] nums, int start, 
                          List<Integer> path, List<List<Integer>> result) {
        // 每個路徑都是一個有效的子集
        result.add(new ArrayList<>(path));
        
        // 嘗試加入後續的每個元素
        for (int i = start; i < nums.length; i++) {
            // 選擇當前元素
            path.add(nums[i]);
            
            // 遞迴處理下一個位置
            backtrack(nums, i + 1, path, result);
            
            // 回溯
            path.remove(path.size() - 1);
        }
    }
}

面試要點

常見追問

  • 如果有重複元素怎麼辦？（Subsets II）
  • 三種方法的優劣比較？
  • 如何只返回長度為 k 的子集？
  • 為什麼有 2^n 個子集？

解題技巧

  • 每個元素都有「選」或「不選」兩種狀態
  • 回溯法從 start 開始避免重複
  • 迭代法思路更直觀
  • 位元操作最簡潔但不易理解

常見錯誤

  • 忘記加入空集
  • 產生重複的子集
  • 沒有複製路徑就加入結果
  • 位元操作時索引錯誤
