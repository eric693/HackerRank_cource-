#136 EASY

Single Number

Array Bit Manipulation

題目描述

給定一個非空整數陣列，除了某個元素只出現一次以外，其餘每個元素均出現兩次。找出那個只出現一次的元素。

說明：你的算法應該具有線性時間複雜度。你可以不使用額外空間來實現嗎？

範例

範例 1:

輸入：nums = [2,2,1]
輸出：1

範例 2:

輸入：nums = [4,1,2,1,2]
輸出：4

範例 3:

輸入：nums = [1]
輸出：1

解題思路

XOR 的性質

  • a ⊕ a = 0
  • a ⊕ 0 = a
  • XOR 滿足交換律和結合律

因此，將所有數字 XOR 在一起，相同的數字會相互抵消，最後剩下的就是只出現一次的數字。

程式碼實作

class Solution:
    def singleNumber(self, nums: List[int]) -> int:
        """
        XOR 位元操作
        時間：O(n), 空間：O(1)
        """
        result = 0
        for num in nums:
            result ^= num
        return result

# 更簡潔的寫法
class Solution2:
    def singleNumber(self, nums: List[int]) -> int:
        from functools import reduce
        return reduce(lambda x, y: x ^ y, nums)

XOR 過程視覺化

例如：[4, 1, 2, 1, 2]

0 ⊕ 4 = 4
4 ⊕ 1 = 5
5 ⊕ 2 = 7
7 ⊕ 1 = 6
6 ⊕ 2 = 4  ← 結果

或用交換律重排：
(4) ⊕ (1 ⊕ 1) ⊕ (2 ⊕ 2) = 4 ⊕ 0 ⊕ 0 = 4

解題技巧

  • XOR 的神奇性質
  • O(n) 時間，O(1) 空間
  • 相同數字 XOR 為 0
  • 任何數與 0 XOR 等於自己
