#15 MEDIUM

3Sum

Array Two Pointers Sorting

題目描述

給定一個整數陣列 nums，找出所有滿足 nums[i] + nums[j] + nums[k] = 0 的三元組 (i, j, k)，其中 i != j、i != k 且 j != k。

注意：答案中不可以包含重複的三元組。

範例

範例 1:

輸入：nums = [-1,0,1,2,-1,-4]
輸出：[[-1,-1,2],[-1,0,1]]
解釋：
nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0
nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0
nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0
不同的三元組是 [-1,0,1] 和 [-1,-1,2]
注意，輸出的順序和三元組的順序並不重要。

範例 2:

輸入：nums = [0,1,1]
輸出：[]
解釋：唯一可能的三元組和不為 0。

範例 3:

輸入：nums = [0,0,0]
輸出：[[0,0,0]]
解釋：唯一可能的三元組和為 0。

限制條件

  • 3 <= nums.length <= 3000
  • -105 <= nums[i] <= 105

解題思路

排序 + 雙指標

這是 Two Sum 的延伸問題，關鍵在於：

  1. 排序陣列：方便去重和使用雙指標
  2. 固定一個數：將 3Sum 轉化為 Two Sum
  3. 雙指標搜尋：在剩餘陣列中找兩數之和
  4. 跳過重複：避免產生重複的三元組

複雜度分析

  • 時間複雜度：O(n²) - 外層迴圈 O(n)，內層雙指標 O(n)
  • 空間複雜度：O(1) 或 O(n) - 取決於排序算法

程式碼實作

Python 解法

class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        """
        排序 + 雙指標解法
        
        時間複雜度：O(n²)
        空間複雜度：O(1)
        """
        nums.sort()
        result = []
        n = len(nums)
        
        # 固定第一個數
        for i in range(n - 2):
            # 跳過重複的第一個數
            if i > 0 and nums[i] == nums[i-1]:
                continue
            
            # 如果最小的數已經大於 0，後面不可能有解
            if nums[i] > 0:
                break
            
            # 使用雙指標找剩餘兩個數
            left = i + 1
            right = n - 1
            
            while left < right:
                total = nums[i] + nums[left] + nums[right]
                
                if total == 0:
                    result.append([nums[i], nums[left], nums[right]])
                    
                    # 跳過重複的第二個數
                    while left < right and nums[left] == nums[left + 1]:
                        left += 1
                    # 跳過重複的第三個數
                    while left < right and nums[right] == nums[right - 1]:
                        right -= 1
                    
                    left += 1
                    right -= 1
                    
                elif total < 0:
                    left += 1
                else:
                    right -= 1
        
        return result

C++ 解法

#include <vector>
#include <algorithm>
using namespace std;

class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        sort(nums.begin(), nums.end());
        vector<vector<int>> result;
        int n = nums.size();
        
        for (int i = 0; i < n - 2; i++) {
            // 跳過重複的第一個數
            if (i > 0 && nums[i] == nums[i-1]) continue;
            
            // 如果最小的數已經大於 0，後面不可能有解
            if (nums[i] > 0) break;
            
            int left = i + 1;
            int right = n - 1;
            
            while (left < right) {
                int total = nums[i] + nums[left] + nums[right];
                
                if (total == 0) {
                    result.push_back({nums[i], nums[left], nums[right]});
                    
                    // 跳過重複的數
                    while (left < right && nums[left] == nums[left + 1]) left++;
                    while (left < right && nums[right] == nums[right - 1]) right--;
                    
                    left++;
                    right--;
                } else if (total < 0) {
                    left++;
                } else {
                    right--;
                }
            }
        }
        
        return result;
    }
};

class Solution {
    public List<List<Integer>> threeSum(int[] nums) {

        int n = nums.length;
        
        for (int i = 0; i < n - 2; i++) {
            // 跳過重複的第一個數
            if (i > 0 && nums[i] == nums[i-1]) continue;

            if (nums[i] > 0) break;
            
            int left = i + 1;
            int right = n - 1;
            
            while (left < right) {
                int total = nums[i] + nums[left] + nums[right];
                
                if (total == 0) {
                    result.add(Arrays.asList(nums[i], nums[left], nums[right]));
                    
                    // 跳過重複的數
                    while (left < right && nums[left] == nums[left + 1]) left++;
                    while (left < right && nums[right] == nums[right - 1]) right--;
                    
                    left++;
                    right--;
                } else if (total < 0) {
                    left++;
                } else {
                    right--;
                }
            }
        }
        
        return result;
    }
}

面試要點

常見追問

  • 如何擴展到 4Sum、kSum？
  • 為什麼要先排序？
  • 如何避免重複的三元組？
  • 能否不排序就解決？

解題技巧

  • 排序是關鍵，方便去重和雙指標
  • 將 3Sum 轉化為 Two Sum
  • 注意跳過重複元素的邏輯
  • 提前終止可以優化效能

常見錯誤

  • 忘記跳過重複元素
  • 邊界條件處理不當
  • 指標移動邏輯錯誤
  • 沒有考慮全為 0 的情況
