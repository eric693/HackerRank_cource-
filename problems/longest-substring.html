#3 MEDIUM

Longest Substring Without Repeating Characters

String Hash Table Sliding Window

題目描述

給定一個字串 s，請找出其中不含有重複字符的最長子字串的長度。

範例

範例 1:

輸入：s = "abcabcbb"
輸出：3
解釋：因為無重複字符的最長子字串是 "abc"，所以其長度為 3。

範例 2:

輸入：s = "bbbbb"
輸出：1
解釋：因為無重複字符的最長子字串是 "b"，所以其長度為 1。

範例 3:

輸入：s = "pwwkew"
輸出：3
解釋：因為無重複字符的最長子字串是 "wke"，所以其長度為 3。
注意：答案必須是子字串，"pwke" 是子序列，不是子字串。

限制條件

  • 0 <= s.length <= 5 * 104
  • s 由英文字母、數字、符號和空格組成

解題思路

滑動視窗（Sliding Window）- 推薦

這是滑動視窗技巧的經典應用：

  1. 使用兩個指標 left 和 right 定義視窗
  2. 使用哈希表記錄字符最後出現的位置
  3. 當遇到重複字符時，移動 left 指標
  4. 不斷更新最大長度
  • 時間複雜度：O(n) - 每個字符最多訪問兩次
  • 空間複雜度：O(min(m, n)) - m 是字符集大小

暴力法（不推薦）

檢查所有子字串，判斷是否有重複字符。

  • 時間複雜度：O(n³)
  • 會超時

程式碼實作

Python 解法

class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        """
        滑動視窗 + 哈希表
        
        時間複雜度：O(n)
        空間複雜度：O(min(m, n))
        """
        # 記錄字符最後出現的位置
        char_index = {}
        max_length = 0
        left = 0
        
        for right in range(len(s)):
            char = s[right]
            
            # 如果字符已經在視窗中，移動左指標
            if char in char_index and char_index[char] >= left:
                left = char_index[char] + 1
            
            # 更新字符位置
            char_index[char] = right
            
            # 更新最大長度
            max_length = max(max_length, right - left + 1)
        
        return max_length


# 方法二：使用 set
class Solution2:
    def lengthOfLongestSubstring(self, s: str) -> int:
        """
        滑動視窗 + Set
        """
        char_set = set()
        left = 0
        max_length = 0
        
        for right in range(len(s)):
            # 當遇到重複字符時，移除左邊的字符
            while s[right] in char_set:
                char_set.remove(s[left])
                left += 1
            
            char_set.add(s[right])
            max_length = max(max_length, right - left + 1)
        
        return max_length


# 方法三：優化的滑動視窗（使用字符陣列）
class Solution3:
    def lengthOfLongestSubstring(self, s: str) -> int:
        """
        使用陣列代替哈希表（ASCII 字符）
        """
        # 假設只有 ASCII 字符（128 個）
        char_index = [-1] * 128
        max_length = 0
        left = 0
        
        for right in range(len(s)):
            char_code = ord(s[right])
            
            # 如果字符在視窗中，移動左指標
            if char_index[char_code] >= left:
                left = char_index[char_code] + 1
            
            char_index[char_code] = right
            max_length = max(max_length, right - left + 1)
        
        return max_length

C++ 解法

#include <string>
#include <unordered_map>
#include <algorithm>
using namespace std;

class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        unordered_map<char, int> charIndex;
        int maxLength = 0;
        int left = 0;
        
        for (int right = 0; right < s.length(); right++) {
            char c = s[right];
            
            // 如果字符已經在視窗中，移動左指標
            if (charIndex.find(c) != charIndex.end() && charIndex[c] >= left) {
                left = charIndex[c] + 1;
            }
            
            charIndex[c] = right;
            maxLength = max(maxLength, right - left + 1);
        }
        
        return maxLength;
    }
};

Java 解法

import java.util.HashMap;
import java.util.Map;

class Solution {
    public int lengthOfLongestSubstring(String s) {
        Map<Character, Integer> charIndex = new HashMap<>();
        int maxLength = 0;
        int left = 0;
        
        for (int right = 0; right < s.length(); right++) {
            char c = s.charAt(right);
            
            // 如果字符已經在視窗中，移動左指標
            if (charIndex.containsKey(c) && charIndex.get(c) >= left) {
                left = charIndex.get(c) + 1;
            }
            
            charIndex.put(c, right);
            maxLength = Math.max(maxLength, right - left + 1);
        }
        
        return maxLength;
    }
}

面試要點

常見追問

  • 如果要求返回子字串本身而非長度？
  • 如果允許最多 k 個重複字符？
  • 空間複雜度能否優化？
  • 為什麼用滑動視窗而不是暴力法？

解題技巧

  • 滑動視窗是處理子字串問題的經典技巧
  • 使用哈希表記錄字符位置
  • 注意左指標的移動條件
  • 理解視窗的伸縮過程

常見錯誤

  • 沒有檢查字符是否在當前視窗內
  • 左指標移動邏輯錯誤
  • 忘記更新字符的最新位置
  • 邊界條件處理不當
