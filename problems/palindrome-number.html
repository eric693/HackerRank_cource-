#9 EASY

Palindrome Number

Math

題目描述

給你一個整數 x，如果 x 是回文整數，返回 true；否則，返回 false。

回文數是指正序和倒序讀都是一樣的整數。例如，121 是回文，而 123 不是。

範例

範例 1:

輸入：x = 121
輸出：true

範例 2:

輸入：x = -121
輸出：false
解釋：從左向右讀為 -121，從右向左讀為 121-，不是回文數。

範例 3:

輸入：x = 10
輸出：false
解釋：從右向左讀為 01，不是回文數。

程式碼實作

class Solution:
    def isPalindrome(self, x: int) -> bool:
        # 負數和以 0 結尾的正數（10, 20...）不是回文
        if x < 0 or (x % 10 == 0 and x != 0):
            return False
        
        # 反轉一半的數字
        reversed_half = 0
        while x > reversed_half:
            reversed_half = reversed_half * 10 + x % 10
            x //= 10
        
        # 奇數長度：x == reversed_half // 10
        # 偶數長度：x == reversed_half
        return x == reversed_half or x == reversed_half // 10
class Solution {
public:
    bool isPalindrome(int x) {
        if (x < 0 || (x % 10 == 0 && x != 0)) return false;
        
        int reversedHalf = 0;
        while (x > reversedHalf) {
            reversedHalf = reversedHalf * 10 + x % 10;
            x /= 10;
        }
        
        return x == reversedHalf || x == reversedHalf / 10;
    }
};
class Solution {
    public boolean isPalindrome(int x) {
        if (x < 0 || (x % 10 == 0 && x != 0)) return false;
        
        int reversedHalf = 0;
        while (x > reversedHalf) {
            reversedHalf = reversedHalf * 10 + x % 10;
            x /= 10;
        }
        
        return x == reversedHalf || x == reversedHalf / 10;
    }
}

面試要點

解題技巧

  • 只需反轉一半即可
  • 負數一定不是回文
  • 以 0 結尾的正數不是回文（除了 0）
  • O(log n) 時間，O(1) 空間
