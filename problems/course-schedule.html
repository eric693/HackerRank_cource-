#207 MEDIUM

Course Schedule

Graph DFS BFS Topological Sort

題目描述

你這個學期必須選修 numCourses 門課程，記為 0 到 numCourses - 1。

在選修某些課程之前需要一些先修課程。先修課程按陣列 prerequisites 給出，其中 prerequisites[i] = [ai, bi] 表示如果要學習課程 ai 則必須先學習課程 bi。

例如，先修課程對 [0, 1] 表示：想要學習課程 0，你需要先完成課程 1。

請你判斷是否可能完成所有課程的學習？如果可以，返回 true；否則，返回 false。

範例

範例 1:

輸入：numCourses = 2, prerequisites = [[1,0]]
輸出：true
解釋：總共有 2 門課程。學習課程 1 之前，你需要完成課程 0。這是可能的。

範例 2:

輸入：numCourses = 2, prerequisites = [[1,0],[0,1]]
輸出：false
解釋：總共有 2 門課程。學習課程 1 之前，你需要先完成課程 0；
     並且學習課程 0 之前，你還應先完成課程 1。這是不可能的。

限制條件

  • 1 <= numCourses <= 2000
  • 0 <= prerequisites.length <= 5000
  • prerequisites[i].length == 2
  • 0 <= ai, bi < numCourses
  • prerequisites[i] 中的所有課程對互不相同

解題思路

核心問題

這是一個檢測有向圖是否有環的問題：

  • 如果圖中有環，則無法完成所有課程（死鎖）
  • 如果圖中無環，則可以完成所有課程

方法一：DFS + 三色標記（推薦）

使用三種狀態標記節點：

  • 白色（0）：未訪問
  • 灰色（1）：正在訪問（在當前路徑中）
  • 黑色（2）：已訪問完成

如果訪問到灰色節點，說明有環。

  • 時間複雜度：O(V + E)
  • 空間複雜度：O(V + E)

方法二：BFS + 拓撲排序（Kahn's Algorithm）

統計每個節點的入度，從入度為 0 的節點開始：

  1. 將所有入度為 0 的節點加入佇列
  2. 取出節點，將其鄰居的入度減 1
  3. 如果鄰居入度變為 0，加入佇列
  4. 最後檢查是否所有節點都被處理
  • 時間複雜度：O(V + E)
  • 空間複雜度：O(V + E)

程式碼實作

Python 解法

class Solution:
    def canFinish(self, numCourses: int, prerequisites: List[List[int]]) -> bool:
        """
        DFS + 三色標記法
        
        時間複雜度：O(V + E)
        空間複雜度：O(V + E)
        """
        # 建立鄰接表
        graph = [[] for _ in range(numCourses)]
        for course, prereq in prerequisites:
            graph[course].append(prereq)
        
        # 0: 未訪問, 1: 正在訪問, 2: 已完成
        visited = [0] * numCourses
        
        def has_cycle(course):
            if visited[course] == 1:
                # 遇到灰色節點，有環
                return True
            if visited[course] == 2:
                # 已經訪問過，無環
                return False
            
            # 標記為正在訪問
            visited[course] = 1
            
            # 訪問所有先修課程
            for prereq in graph[course]:
                if has_cycle(prereq):
                    return True
            
            # 標記為已完成
            visited[course] = 2
            return False
        
        # 檢查每門課程
        for i in range(numCourses):
            if has_cycle(i):
                return False
        
        return True


# 方法二：BFS + 拓撲排序
from collections import deque

class Solution2:
    def canFinish(self, numCourses: int, prerequisites: List[List[int]]) -> bool:
        """
        BFS + Kahn's Algorithm
        
        時間複雜度：O(V + E)
        空間複雜度：O(V + E)
        """
        # 建立鄰接表和入度陣列
        graph = [[] for _ in range(numCourses)]
        in_degree = [0] * numCourses
        
        for course, prereq in prerequisites:
            graph[prereq].append(course)
            in_degree[course] += 1
        
        # 將所有入度為 0 的課程加入佇列
        queue = deque([i for i in range(numCourses) if in_degree[i] == 0])
        
        # 記錄已處理的課程數
        count = 0
        
        while queue:
            course = queue.popleft()
            count += 1
            
            # 處理該課程的後續課程
            for next_course in graph[course]:
                in_degree[next_course] -= 1
                if in_degree[next_course] == 0:
                    queue.append(next_course)
        
        # 如果所有課程都被處理，說明無環
        return count == numCourses

C++ 解法

#include <vector>
#include <queue>
using namespace std;

class Solution {
public:
    bool canFinish(int numCourses, vector<vector<int>>& prerequisites) {
        // 建立鄰接表
        vector<vector<int>> graph(numCourses);
        for (auto& p : prerequisites) {
            graph[p[0]].push_back(p[1]);
        }
        
        // 0: 未訪問, 1: 正在訪問, 2: 已完成
        vector<int> visited(numCourses, 0);
        
        // 檢查每門課程
        for (int i = 0; i < numCourses; i++) {
            if (hasCycle(graph, visited, i)) {
                return false;
            }
        }
        
        return true;
    }
    
private:
    bool hasCycle(vector<vector<int>>& graph, vector<int>& visited, int course) {
        if (visited[course] == 1) return true;
        if (visited[course] == 2) return false;
        
        visited[course] = 1;
        
        for (int prereq : graph[course]) {
            if (hasCycle(graph, visited, prereq)) {
                return true;
            }
        }
        
        visited[course] = 2;
        return false;
    }
};

Java 解法

import java.util.*;

class Solution {
    public boolean canFinish(int numCourses, int[][] prerequisites) {
        // 建立鄰接表
        List<List<Integer>> graph = new ArrayList<>();
        for (int i = 0; i < numCourses; i++) {
            graph.add(new ArrayList<>());
        }
        
        for (int[] p : prerequisites) {
            graph.get(p[0]).add(p[1]);
        }
        
        // 0: 未訪問, 1: 正在訪問, 2: 已完成
        int[] visited = new int[numCourses];
        
        // 檢查每門課程
        for (int i = 0; i < numCourses; i++) {
            if (hasCycle(graph, visited, i)) {
                return false;
            }
        }
        
        return true;
    }
    
    private boolean hasCycle(List<List<Integer>> graph, int[] visited, int course) {
        if (visited[course] == 1) return true;
        if (visited[course] == 2) return false;
        
        visited[course] = 1;
        
        for (int prereq : graph.get(course)) {
            if (hasCycle(graph, visited, prereq)) {
                return true;
            }
        }
        
        visited[course] = 2;
        return false;
    }
}

面試要點

常見追問

  • 如何返回一個有效的課程順序？（Course Schedule II）
  • DFS 和 BFS 哪個更好？
  • 什麼是拓撲排序？
  • 如何檢測無向圖的環？

解題技巧

  • 識別出這是有向圖環檢測問題
  • 三色標記法是經典 DFS 技巧
  • 拓撲排序適合依賴關係問題
  • 建圖是關鍵第一步

常見錯誤

  • 建圖方向錯誤
  • 只用兩種狀態無法檢測環
  • 忘記重置訪問狀態
  • BFS 中入度計算錯誤

圖解說明

範例：numCourses = 4, prerequisites = [[1,0],[2,0],[3,1],[3,2]]

依賴關係圖：
    0
   / \
  1   2
   \ /
    3

拓撲排序順序：0 → 1 → 2 → 3 （或 0 → 2 → 1 → 3）
可以完成所有課程 ✓

有環的例子：[[1,0],[0,1]]
    0 ←→ 1
無法完成 ✗
