#128 MEDIUM

Longest Consecutive Sequence

Array Hash Table Union Find

題目描述

給定一個未排序的整數陣列 nums，找出數字連續的最長序列（不要求序列元素在原陣列中連續）的長度。

請你設計並實現時間複雜度為 O(n) 的算法解決此問題。

範例

範例 1:

輸入：nums = [100,4,200,1,3,2]
輸出：4
解釋：最長數字連續序列是 [1, 2, 3, 4]。它的長度為 4。

範例 2:

輸入：nums = [0,3,7,2,5,8,4,6,0,1]
輸出：9
解釋：最長連續序列是 [0,1,2,3,4,5,6,7,8]。

限制條件

  • 0 <= nums.length <= 105
  • -109 <= nums[i] <= 109

解題思路

哈希集合法（推薦）- O(n)

關鍵觀察：只從序列的起點開始計數

算法步驟：

  1. 將所有數字放入哈希集合中
  2. 遍歷每個數字 num：
    • 如果 num - 1 在集合中，跳過（不是起點）
    • 如果 num - 1 不在集合中，從 num 開始計數
  3. 持續檢查 num+1, num+2, ... 是否在集合中
  4. 記錄最長序列長度

為什麼是 O(n)？

雖然有嵌套循環，但每個數字最多被訪問兩次：

  • 一次在外層循環中檢查是否為起點
  • 一次在內層循環中作為序列的一部分
  • 時間複雜度：O(n)
  • 空間複雜度：O(n)

程式碼實作

Python 解法

class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        """
        哈希集合 O(n) 解法
        
        時間複雜度：O(n)
        空間複雜度：O(n)
        """
        if not nums:
            return 0
        
        # 將所有數字放入集合
        num_set = set(nums)
        longest = 0
        
        for num in num_set:
            # 只從序列起點開始計數
            # 如果 num-1 存在，說明不是起點，跳過
            if num - 1 not in num_set:
                current_num = num
                current_length = 1
                
                # 向後延伸序列
                while current_num + 1 in num_set:
                    current_num += 1
                    current_length += 1
                
                longest = max(longest, current_length)
        
        return longest


# 方法二：排序後掃描（O(n log n)）
class Solution2:
    def longestConsecutive(self, nums: List[int]) -> int:
        """
        排序解法（不符合 O(n) 要求，但思路簡單）
        
        時間複雜度：O(n log n)
        空間複雜度：O(1)
        """
        if not nums:
            return 0
        
        nums.sort()
        longest = 1
        current_length = 1
        
        for i in range(1, len(nums)):
            if nums[i] == nums[i-1]:
                continue  # 跳過重複
            elif nums[i] == nums[i-1] + 1:
                current_length += 1
                longest = max(longest, current_length)
            else:
                current_length = 1
        
        return longest

C++ 解法

#include <vector>
#include <unordered_set>
#include <algorithm>
using namespace std;

class Solution {
public:
    int longestConsecutive(vector<int>& nums) {
        if (nums.empty()) return 0;
        
        // 將所有數字放入集合
        unordered_set<int> numSet(nums.begin(), nums.end());
        int longest = 0;
        
        for (int num : numSet) {
            // 只從序列起點開始
            if (numSet.find(num - 1) == numSet.end()) {
                int currentNum = num;
                int currentLength = 1;
                
                // 向後延伸序列
                while (numSet.find(currentNum + 1) != numSet.end()) {
                    currentNum++;
                    currentLength++;
                }
                
                longest = max(longest, currentLength);
            }
        }
        
        return longest;
    }
};

Java 解法

import java.util.*;

class Solution {
    public int longestConsecutive(int[] nums) {
        if (nums.length == 0) return 0;
        
        // 將所有數字放入集合
        Set<Integer> numSet = new HashSet<>();
        for (int num : nums) {
            numSet.add(num);
        }
        
        int longest = 0;
        
        for (int num : numSet) {
            // 只從序列起點開始
            if (!numSet.contains(num - 1)) {
                int currentNum = num;
                int currentLength = 1;
                
                // 向後延伸序列
                while (numSet.contains(currentNum + 1)) {
                    currentNum++;
                    currentLength++;
                }
                
                longest = Math.max(longest, currentLength);
            }
        }
        
        return longest;
    }
}

算法過程視覺化

以 nums = [100, 4, 200, 1, 3, 2] 為例：

步驟 1：建立集合
集合: {100, 4, 200, 1, 3, 2}

步驟 2：遍歷每個數字

檢查 100:
  99 不在集合中 → 是起點
  檢查 101... 不在
  序列長度 = 1

檢查 4:
  3 在集合中 → 不是起點，跳過

檢查 200:
  199 不在集合中 → 是起點
  檢查 201... 不在
  序列長度 = 1

檢查 1:
  0 不在集合中 → 是起點 ✓
  檢查 2 → 在 ✓
  檢查 3 → 在 ✓
  檢查 4 → 在 ✓
  檢查 5 → 不在
  序列長度 = 4 (最長) ✓

檢查 3:
  2 在集合中 → 不是起點，跳過

檢查 2:
  1 在集合中 → 不是起點，跳過

最終答案 = 4

面試要點

常見追問

  • 為什麼時間複雜度是 O(n)？
  • 能否不使用額外空間？
  • 如何處理重複元素？
  • Union Find 如何實現？

解題技巧

  • 只從序列起點開始是關鍵
  • 哈希集合實現 O(1) 查找
  • 每個數字最多被訪問 2 次
  • 重複元素會被集合自動去除

常見錯誤

  • 忘記檢查是否為序列起點
  • 使用排序導致 O(n log n)
  • 重複計算同一序列
  • 忘記處理空陣列情況
